## Overview

Returns all master address records associated with a specific master company or master person.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="master_person_id" type="integer">
  Filter addresses by master person ID.
</ParamField>

<ParamField query="master_company_id" type="integer">
  Filter addresses by master company ID.
</ParamField>

## Response

Returns an array of master address records.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:Lnhgpj_M/master-addresses?master_company_id=100" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "id": 1,
    "master_company_id": 100,
    "address_label": "Headquarters",

  }
```

</ResponseExample>

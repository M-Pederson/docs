---

## Overview

Assigns a collection node to a specific user.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_node_id" type="integer" required>
  The ID of the collection node to assign.
</ParamField>

## Request Body

<ParamField body="collection_node_id" type="integer" required>
  The ID of the collection node.
</ParamField>

<ParamField body="user_id" type="integer" required>
  The ID of the user to assign the node to.
</ParamField>

## Response

Returns the assignment result.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes/1/assign \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_node_id": 1,
    "user_id": 10
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_node_id": 1,
  "user_id": 10
}
```
</ResponseExample>

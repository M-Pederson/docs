## Overview

Moves one or more collection nodes from their current collection to a different collection.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="collection_node_ids" type="array" required>
  Array of collection node IDs to move.
</ParamField>

<ParamField body="collection_id" type="integer" required>
  The destination collection ID to move the nodes to.
</ParamField>

## Response

Returns the result of the move operation.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes/move \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_node_ids": [1, 2, 3],
    "collection_id": 5
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "moved": 3
}
```
</ResponseExample>

title: 'Create Master Company'
api: 'POST https://xh2o-yths-38lt.n7c.xano.io/api:10Em-CSF/master-companies'
description: 'Create a new master company record and my_company entry'
---

## Overview

Creates a new master company record and automatically adds it to the authenticated user's company list (my_company).

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="company_name" type="string" required>
  The name of the company.
</ParamField>

<ParamField body="domain" type="string" required>
  The company's website domain.
</ParamField>

## Response

Returns the created master company record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:10Em-CSF/master-companies \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "company_name": "Acme Corp",
    "domain": "acme.com"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 100,
  "company_name": "Acme Corp",
  "domain": "acme.com"
}
```
</ResponseExample>

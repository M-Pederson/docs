---
title: "Create Master Phone"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:KJyr4R-l/master-phones"
description: "Add a master phone record for either a company or person."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

### Body Parameters

<ParamField body="master_company_id" type="integer">
  The master company ID to associate with. Nullable.
</ParamField>

<ParamField body="master_person_id" type="integer">
  The master person ID to associate with. Nullable.
</ParamField>

<ParamField body="phone_number" type="string">
  The phone number.
</ParamField>

<ParamField body="phone_extension" type="integer">
  Phone extension. Nullable.
</ParamField>

<ParamField body="phone_type" type="string">
  Type of phone. One of: `mobile`, `office`, `other`, `home`. Defaults to `mobile`.
</ParamField>

<ParamField body="phone_note" type="string">
  Optional note for the phone record. Nullable.
</ParamField>

### Response

Returns the created master phone record.

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:KJyr4R-l/master-phones' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "master_person_id": 1,
    "phone_number": "+1-555-123-4567",
    "phone_type": "mobile"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

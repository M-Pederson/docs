## Overview

Permanently deletes a collection and its associated data.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_id" type="integer" required>
  The ID of the collection to delete.
</ParamField>

## Response

Returns an empty object on success.

<RequestExample>
```bash cURL
curl -X DELETE https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collections/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

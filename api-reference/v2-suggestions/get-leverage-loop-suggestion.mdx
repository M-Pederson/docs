## Overview

Retrieve a single leverage loop suggestion record by its ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField body="leverage_loop_suggestion_id" type="integer" required>
  The ID of the leverage loop suggestion to retrieve.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The leverage loop suggestion ID.
</ResponseField>

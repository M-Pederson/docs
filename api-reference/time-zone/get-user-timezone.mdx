---
title: "Get User Timezone"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:0sJ9WWVZ/get/user-timezone"
description: "Resolve and return the user's timezone based on browser timezone input."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="browser_timezone" type="string">
  The browser-detected timezone string (e.g. `America/New_York`).
</ParamField>

<ResponseField name="time_zone" type="string">
  The resolved timezone for the user.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:0sJ9WWVZ/get/user-timezone' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "browser_timezone": "America/New_York"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "time_zone": "America/New_York"
}
```
</ResponseExample>

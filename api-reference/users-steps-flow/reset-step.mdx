---
title: "Reset Step"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:_wjCLRXt/new-user/step/reset"
description: "Reset the user's onboarding step to a specific value."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="step" type="integer">The step number to reset to.</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:_wjCLRXt/new-user/step/reset' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"step": 2}'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "steps": 2,
  "orbiter_url": "johndoe"
}
```
</ResponseExample>

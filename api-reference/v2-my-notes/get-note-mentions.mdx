---
title: "Get Note Mentions"
api: "GET https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/{my_note_id}/mentions"
description: "Retrieves all Orbiter-connected mentions for a specific note."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField path="my_note_id" type="integer" required>
  The note ID to retrieve mentions for.
</ParamField>

<ResponseField name="items" type="array">
  List of mentioned entities.
  <Expandable>
    <ResponseField name="id" type="integer">
      ID of the mentioned entity.
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the mentioned entity.
    </ResponseField>
    <ResponseField name="avatar" type="string">
      Avatar URL of the mentioned entity.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/1/mentions' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "id": 5,
    "name": "John Doe",
    "avatar": "https://example.com/avatar.jpg"
  }
]
```
</ResponseExample>

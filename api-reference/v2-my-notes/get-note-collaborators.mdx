---
title: "Get Note Collaborators"
api: "GET https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborators"
description: "Query all collaborators for a specific note."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField query="my_note_id" type="integer" required>
  The note ID to list collaborators for.
</ParamField>

<ResponseField name="items" type="array">
  List of collaborator records.
  <Expandable>
    <ResponseField name="id" type="integer">Collaborator record ID.</ResponseField>
    <ResponseField name="my_note_id" type="integer">The shared note ID.</ResponseField>
    <ResponseField name="shared_with_user" type="integer">User ID the note is shared with.</ResponseField>
    <ResponseField name="permission_level" type="string">Access level: `editor`, `viewer`, or `commentor`.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborators?my_note_id=1' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

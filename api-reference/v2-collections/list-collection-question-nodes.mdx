## Overview

Returns a paginated list of all question node records associated with a specific collection question.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="collection_question_id" type="integer" required>
  The ID of the collection question to get nodes for.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer">
  Number of items per page.
</ParamField>

<ParamField query="offset" type="integer">
  Number of items to skip.
</ParamField>

## Response

Returns a paginated list of question node records.


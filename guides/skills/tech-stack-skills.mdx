## Overview

Tech Stack Skills are third-party [Claude agent skills](https://code.claude.com/docs/en/skills) that target the specific frameworks and services Orbiter.io depends on. Install them into `~/.claude/skills/` so agents reason fluently about these tools without being re-taught the fundamentals each session.

## Installed skills

### FalkorDB

Practical [FalkorDB](https://www.falkordb.com) guidance bundling 27 runnable skills across Cypher queries, user-defined functions, and Docker operations. Activates when writing or reviewing FalkorDB queries, setting up containers, or managing UDF libraries.

- **Source:** [FalkorDB/skills](https://github.com/FalkorDB/skills)
- **Install:**
  ```bash
  git clone https://github.com/FalkorDB/skills ~/.claude/skills/falkordb
  ```

### Tiptap

Official [Tiptap](https://tiptap.dev) agent skill from the Tiptap team (Ueberdosis). Helps agents integrate the Tiptap rich text editor — installing extensions, building editors with React, implementing collaboration, comments, AI features, and document conversion.

- **Source:** [ueberdosis/tiptap](https://github.com/ueberdosis/tiptap/blob/main/skills/tiptap/SKILL.md) (official)
- **Install:**
  ```bash
  mkdir -p ~/.claude/skills/tiptap
  curl -sL https://raw.githubusercontent.com/ueberdosis/tiptap/main/skills/tiptap/SKILL.md \
    -o ~/.claude/skills/tiptap/SKILL.md
  ```

### TanStack Start

Best practices for [TanStack Start](https://tanstack.com/start) full-stack React applications — server functions, middleware, SSR, authentication, and deployment patterns. Pulled from the broader TanStack agent skills bundle (also includes Query, Router, and integration skills if desired).

- **Source:** [DeckardGer/tanstack-agent-skills](https://github.com/DeckardGer/tanstack-agent-skills)
- **Install:**
  ```bash
  git clone https://github.com/DeckardGer/tanstack-agent-skills /tmp/tanstack-skills
  mv /tmp/tanstack-skills/skills/tanstack-start ~/.claude/skills/tanstack-start
  rm -rf /tmp/tanstack-skills
  ```

### v0-dev

[v0](https://v0.dev) by Vercel is an AI-powered development agent that converts natural language and visual inputs into production-ready frontend code. Generates React components with shadcn/ui + Tailwind by default, supports multi-framework output (React, Vue, Svelte, HTML), and handles Figma/screenshot/sketch visual input.

- **Source:** [vercel-labs/skills](https://github.com/vercel-labs/skills) (official Vercel skills ecosystem)
- **Install:**
  ```bash
  npx skills add vercel-labs/skills --skill v0-dev --agent claude-code --yes
  ```

### Frontend Design

Official Anthropic skill for creating distinctive, production-grade frontend interfaces. Guides agents to avoid generic AI aesthetics — covers tone and typography (picking extreme aesthetic directions, unique font pairings), color and theme (cohesive palettes with CSS variables), and motion (CSS-only animations, staggered reveals, micro-interactions).

- **Source:** [anthropics/claude-code](https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design/skills/frontend-design) (official)
- **Install:**
  ```bash
  mkdir -p ~/.claude/skills/frontend-design
  curl -sL https://raw.githubusercontent.com/anthropics/claude-code/main/plugins/frontend-design/skills/frontend-design/SKILL.md \
    -o ~/.claude/skills/frontend-design/SKILL.md
  ```

### UI UX Pro Max

Comprehensive design intelligence skill with 161 industry-specific rules, 67 UI styles (Glassmorphism, Claymorphism, AI-Native UI, etc.), 161 color palettes, 57 font pairings, 25 chart type recommendations, and 99 UX guidelines with accessibility rules. Supports 15 tech stacks including React, Next.js, Vue, Svelte, SwiftUI, and Flutter.

- **Source:** [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)
- **Install:**
  ```bash
  npx skills add nextlevelbuilder/ui-ux-pro-max-skill --agent claude-code --yes
  ```

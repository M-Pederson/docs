## All Functions

### Person Pipeline — `get-add/master-person` (function #12553)

| Function | ID | Nodes / Edges |
|----------|----|---------------|
| `mvp/node/add-person-node` | #2601 | CREATE `Person` node |
| `mvp/node/update-person-node` | #2758 | SET properties + embedding, SET `Angel` or `Person` label |
| `mvp/edges/create-work-edges` | #2800 | MERGE `WORKS_AT`, `FOUNDED`, `BOARD_MEMBER_OF`, `ADVISOR_TO`, `INVESTED_IN`, `MEMBER_OF` |
| `mvp/edges/create-education-edges` | #1898 | MERGE `ATTENDED`, `ATTENDED_TOGETHER`, `STUDIED_UNDER` |
| `mvp/resolve/resolve-edges-certifications` | #12578 | MERGE `Certification` node + `HAS_CERTIFICATION`, `GRANTED_BY`, `CERTIFIED_BY` |
| `mvp/resolve/resolve-edges-honor` | #4574 | CREATE `Honor` node + `RECEIVED_HONOR`, `ISSUED_BY` |
| `mvp/resolve/resolve-edges-projects-publications` | #12579 | CREATE `Project` + `CONTRIBUTED_TO`, CREATE `Publication` + `AUTHORED`, `PUBLISHED_BY` |
| `mvp/resolve/resolve-edges-volunteering` | #4575 | MERGE `Organization` + CREATE `VOLUNTEERED_AT` |
| `mvp/node/primary-location-nodes-edges_v2` | #4690 | MERGE `Country`, `Region`, `City` + `LOCATED_IN`, `IN_REGION`, `IN_COUNTRY` |

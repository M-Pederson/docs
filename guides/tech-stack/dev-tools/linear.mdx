## What it is

[Linear](https://linear.app) is a project management and issue tracking tool built for software teams. It provides a fast, keyboard-driven interface for managing issues, projects, cycles, and roadmaps.

## Why we use it

- **Speed** — sub-100ms UI, keyboard-first workflow, minimal friction
- **Cycles & projects** — time-boxed sprints and long-running project tracking
- **Git integration** — branch names auto-link to issues, PRs update status
- **API & webhooks** — full GraphQL API for automation and integrations
- **Triage workflow** — inbox-style triage for incoming issues and requests

## How we use it at Orbiter

Linear is our primary engineering project management tool. All feature work, bug fixes, and infrastructure tasks are tracked as Linear issues. We use cycles for sprint planning and projects for larger initiatives.

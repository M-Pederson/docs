---

## Overview

Updates the text of an existing collection comment.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_comment_id" type="integer" required>
  The ID of the comment to update.
</ParamField>

## Request Body

<ParamField body="collection_comment_id" type="integer" required>
  The ID of the comment.
</ParamField>

<ParamField body="comment" type="string" required>
  The updated comment text.
</ParamField>

## Response

Returns the updated comment record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-comments/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_comment_id": 1,
    "comment": "Updated comment text"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "comment": "Updated comment text"
}
```
</ResponseExample>

## Overview

Tech Stack Skills are third-party [Claude agent skills](https://code.claude.com/docs/en/skills) that target the specific frameworks and services Orbiter.io depends on. Install them into `~/.claude/skills/` so agents reason fluently about these tools without being re-taught the fundamentals each session.

## Installed skills

### FalkorDB

Practical [FalkorDB](https://www.falkordb.com) guidance bundling 27 runnable skills across Cypher queries, user-defined functions, and Docker operations. Activates when writing or reviewing FalkorDB queries, setting up containers, or managing UDF libraries.

- **Source:** [FalkorDB/skills](https://github.com/FalkorDB/skills)
- **Install:**
  ```bash
  git clone https://github.com/FalkorDB/skills ~/.claude/skills/falkordb
  ```

### Tiptap

Official [Tiptap](https://tiptap.dev) agent skill from the Tiptap team (Ueberdosis). Helps agents integrate the Tiptap rich text editor — installing extensions, building editors with React, implementing collaboration, comments, AI features, and document conversion.

- **Source:** [ueberdosis/tiptap](https://github.com/ueberdosis/tiptap/blob/main/skills/tiptap/SKILL.md) (official)
- **Install:**
  ```bash
  mkdir -p ~/.claude/skills/tiptap
  curl -sL https://raw.githubusercontent.com/ueberdosis/tiptap/main/skills/tiptap/SKILL.md \
    -o ~/.claude/skills/tiptap/SKILL.md
  ```

### TanStack Start

Best practices for [TanStack Start](https://tanstack.com/start) full-stack React applications — server functions, middleware, SSR, authentication, and deployment patterns. Pulled from the broader TanStack agent skills bundle (also includes Query, Router, and integration skills if desired).

- **Source:** [DeckardGer/tanstack-agent-skills](https://github.com/DeckardGer/tanstack-agent-skills)
- **Install:**
  ```bash
  git clone https://github.com/DeckardGer/tanstack-agent-skills /tmp/tanstack-skills
  mv /tmp/tanstack-skills/skills/tanstack-start ~/.claude/skills/tanstack-start
  rm -rf /tmp/tanstack-skills
  ```

## Overview

The [Graphistry](https://www.graphistry.com) skills package provides context-aware guidance for GPU-accelerated graph analytics and visualization using [PyGraphistry](https://github.com/graphistry/pygraphistry). These skills improve AI success rates on PyGraphistry tasks from ~50% to ~90% by teaching agents the correct patterns for graph ETL, visualization bindings, GFQL queries, and AI workflows.

All skills are installed from the official [graphistry/graphistry-skills](https://github.com/graphistry/graphistry-skills) repository.

## Install

**Full install (recommended):**

```bash
npx skills add graphistry/graphistry-skills \
  --agent claude-code \
  --skill graphistry \
  --skill graphistry-rest-api \
  --skill pygraphistry \
  --skill pygraphistry-core \
  --skill pygraphistry-gfql \
  --skill pygraphistry-visualization \
  --skill pygraphistry-ai \
  --skill pygraphistry-connectors \
  --yes
```

**Minimal REST-only install:**

```bash
npx skills add graphistry/graphistry-skills \
  --agent claude-code \
  --skill graphistry \
  --skill graphistry-rest-api \
  --yes
```

## Skills

### graphistry

Umbrella router across all Graphistry interfaces — SDK and REST. Routes to the appropriate sub-skill based on the task at hand.

### graphistry-rest-api

REST specialist covering authentication, upload lifecycle, URL controls, sessions, and sharing safety. Use when interacting with the Graphistry server API directly rather than through the Python SDK.

### pygraphistry

Python SDK router. Activates on any PyGraphistry task and delegates to the appropriate sub-skill below.

### pygraphistry-core

Authentication, data shaping, and initial visualization workflows. The foundation for all PyGraphistry work — covers `graphistry.edges()`, `graphistry.nodes()`, `graphistry.bind()`, and `graphistry.plot()`.

### pygraphistry-visualization

Bindings, encodings, layout, privacy, and sharing patterns. Covers visual encoding (color, size, icons), layout algorithms, scene settings, and URL sharing configuration.

### pygraphistry-gfql

GFQL (Graph Frame Query Language) extraction and pattern workflows. Covers chain-list syntax, Cypher string queries, Let/DAG bindings, GRAPH constructors, and remote execution.

### pygraphistry-ai

Embedding, UMAP/DBSCAN, and anomaly detection workflows. Covers `g.umap()`, `g.dbscan()`, feature engineering, and the AI pipeline for graph-native ML.

### pygraphistry-connectors

Connector and integration workflows. Covers loading data from various sources (Neo4j, Splunk, Neptune, TigerGraph, etc.) into PyGraphistry.

---

## Overview

Updates the text of an existing collection question.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_question_id" type="integer" required>
  The ID of the question to update.
</ParamField>

## Request Body

<ParamField body="question" type="string" required>
  The updated question text.
</ParamField>

## Response

Returns the updated question record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-questions/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "question": "Who are the key decision makers?"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "question": "Who are the key decision makers?"
}
```
</ResponseExample>

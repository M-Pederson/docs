## What it is

[Tavily](https://tavily.com) is a search API built specifically for AI agents and LLM applications. It returns clean, relevant results optimized for consumption by language models rather than raw HTML pages.

## Why we use it

- **AI-native search** — results are pre-processed and structured for LLM consumption
- **Fast and relevant** — returns concise, high-quality answers and source URLs
- **Simple API** — single endpoint with straightforward request/response format
- **Content extraction** — can return full page content alongside search results

## How we use it at Orbiter

Tavily is one of our web search providers, used in enrichment pipelines where we need to look up real-time information from the web to supplement our data.

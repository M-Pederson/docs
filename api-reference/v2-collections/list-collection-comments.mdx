## Overview

Returns all comments posted to a specific collection.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="collection_id" type="integer" required>
  The ID of the collection to get comments for.
</ParamField>

## Response

Returns an array of comment records.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-comments?collection_id=1" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "id": 1,
    "collection_id": 1,
    "comment": "Great leads in this collection!",
    "user_id": 10,
    "created_at": 1700000000
  }
]
```
</ResponseExample>

## Overview

Deletes a master address record by its ID. This also removes any associated address notes and join records.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="master_address_id" type="integer" required>
  The ID of the master address to delete.
</ParamField>

## Response

Returns an empty object on success.

<RequestExample>
```bash cURL
curl -X DELETE https://xh2o-yths-38lt.n7c.xano.io/api:Lnhgpj_M/master-addresses/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

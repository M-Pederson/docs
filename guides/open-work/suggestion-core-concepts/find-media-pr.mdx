## Example Final Dispatch

The payload the copilot emits after the interview ends — handed off to the downstream
matching agent. Modeled on Example 1 (Series A out-of-stealth, AI infra) below.

Top-level fields are required. Everything under `media_spec` is the structured intake from
the chat — every dimension a downstream agent needs to score journalist matches is included here.

```json

  "user_id": "mp_8f3a91",

  "number_of_results": 7,
  "format_of_results": "suggestion",

  "media_spec": {
    "story_type": "funding_announcement",
    "story_peg": "Series A out-of-stealth, $18M from tier-1 fund — first public moment for the company",
    "angle": "Technical differentiation in AI infra, not a paint-by-numbers funding story",

    "media_tier": ["tier_1_national_tech", "vertical_trade_ai_infra"],
    "formats_accepted": ["long_form_article", "exclusive_scoop", "newsletter_feature"],

    "target_audience_reached": ["enterprise engineering buyers", "developers", "technical investors"],

    "exclusivity": {
      "offer_type": "exclusive_first_right",

      "embargo_lift_date": "2026-05-14T09:00:00-04:00",

    "warm_path": {
      "preference": "warm_intro_preferred",
      "cold_pitch_acceptable": true,
      "max_hops": 2
    },

    "must_haves": [
      "Has covered AI infrastructure (not just AI broadly) in the last 6 months",
      "Writes original analysis, not aggregator-style rewrites",

    "nice_to_haves": [

    "deal_breakers": [
      "Generalist tech reporter with no infrastructure coverage",
      "Outlet primarily covering consumer tech or lifestyle",
      "Recently covered a direct competitor in the last 3 months"
    ],

    "exclusion_list": {
      "master_person_ids": [],
      "outlets": ["TechCrunch aggregator desk"],
      "notes": "No rewrite desks — must be byline-accountable journalists"
    }
  },

  "matching_hints": {
    "rank_by": ["beat_match", "recency_of_coverage", "audience_reach", "warm_path_distance"],
    "warm_path_required": false,
    "warm_path_max_hops": 2,
    "fresh_signal_window_days": 180
  },

  "dispatch": {
    "agent": "match-people-v2",
    "callback_route": "/copilot/outcomes/find_media_pr/results",
    "created_at": "2026-05-07T14:32:11Z",
    "outcome_id": "out_01J9ZK4B6T9X3M2N7Q5P8R0V1W"
```

**Notes for downstream agents**

- `outcome_summary` is the human-readable one-liner; `media_spec` is the structured truth.
  Score on `media_spec`, render on `outcome_summary`.
- `beat_match.required_beats` are filters — a journalist with no coverage on any required beat
  is excluded. `preferred_beats` are ranking signals only.
- `beat_match.recency_signal` is a hard freshness gate: stale beat coverage from 18+ months
  ago does not qualify. Use the `fresh_signal_window_days` in `matching_hints` to enforce.
- `exclusivity.offer_type` must be respected when ranking: if `exclusive_first_right`, surface
  only one outlet/journalist as primary; remaining slots are second-wave candidates.
- `must_haves` are hard filters. `deal_breakers` are disqualifying signals — apply before scoring.
- `exclusion_list.outlets` blocks the entire outlet, not just specific journalists within it.
- `format_of_results` is one of `suggestion | csv | pdf`. `suggestion` returns enriched
  journalist cards into the copilot UI; `csv`/`pdf` are operator-facing exports for PR handoff.

## Description

This outcome type surfaces specific journalists, podcasters, newsletter writers, and producers who cover the User's space. The goal is press coverage — by name, not by outlet — when blasting a press release won't work.

**Best for:** founder-led PR, book launches, fund/portfolio announcements, milestone moments.

**Adjacent outcomes:** [Find speakers / get speaking opportunities](/guides/open-work/suggestion-core-concepts/find-speakers) when the platform is stage instead of ink, [Identify strategic partnerships](/guides/open-work/suggestion-core-concepts/strategic-partnerships) for co-marketing relationships.

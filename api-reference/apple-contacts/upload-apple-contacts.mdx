## Overview

Upload Apple contacts by providing vCard data or a vCard file. The contacts will be processed and added to the system.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="data" type="string">
  The vCard data as a string.
</ParamField>

<ParamField body="input_resource" type="file">
  A vCard file to upload.
</ParamField>

## Response

Returns a confirmation object.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:4qGscqU8/apple_contacts \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "input_resource=@contacts.vcf"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

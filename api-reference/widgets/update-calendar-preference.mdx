title: "Update Calendar Preference"

description: "Toggle whether a calendar is active and visible in widgets."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="calendar_id" type="integer" required>The calendar ID.</ParamField>
<ParamField body="active" type="boolean" required>Whether the calendar should be active.</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:3IvQ7jxX/widget/calendar/preferences/update' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"calendar_id": 1, "active": true}'
```
</RequestExample>

<ResponseExample>

  "id": 1,
  "name": "My Calendar",
  "event_widget_preference": true

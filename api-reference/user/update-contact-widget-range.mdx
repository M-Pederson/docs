description: "Update the contact widget date range setting (1, 7, or 30 days)."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="range" type="integer" required>
  The date range in days. Accepted values: `1`, `7`, or `30`.
</ParamField>

<ResponseField name="settings" type="object">
  The updated user settings object.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH \
  https://xh2o-yths-38lt.n7c.xano.io/api:n2ii7Rpf/settings/contact_widget_range/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: live" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "range": 7
  }'
```
</RequestExample>

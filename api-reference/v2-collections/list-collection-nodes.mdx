## Overview

Returns a paginated list of all nodes (persons, companies, funding rounds, etc.) within a specific collection.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="collection_id" type="integer" required>
  The ID of the collection to get nodes for.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="offset" type="integer">
  Number of items to skip.
</ParamField>

<ParamField query="limit" type="integer">
  Number of items per page.
</ParamField>

## Response

Returns a paginated list of collection node records.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes?collection_id=1&limit=50" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

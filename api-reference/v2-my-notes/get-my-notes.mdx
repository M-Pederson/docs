---
title: "Get My Notes"
api: "GET https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes"
description: "Get notes for either my_company or my_person for the authenticated user."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField query="my_company_id" type="integer">
  Filter notes by company ID.
</ParamField>
<ParamField query="my_person_id" type="integer">
  Filter notes by person ID.
</ParamField>

<ResponseField name="items" type="array">
  List of note objects belonging to the authenticated user.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes?my_company_id=1' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

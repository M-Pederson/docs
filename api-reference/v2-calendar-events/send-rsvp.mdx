---
title: "Send RSVP"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:LR2ywW7R/calendar-events/send-rsvp"
description: "Send an RSVP response to a calendar event."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

### Body Parameters

<ParamField body="event_id" type="string" required>
  The field ID in the Nylas response for events.
</ParamField>

<ParamField body="calendar_id" type="string" required>
  The calendar ID the event belongs to.
</ParamField>

<ParamField body="grant_id" type="string" required>
  The Nylas grant ID for the calendar account.
</ParamField>

<ParamField body="status" type="string" required>
  RSVP status. Must be one of: `yes`, `no`, `maybe`.
</ParamField>

### Response

Returns the RSVP confirmation object.

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:LR2ywW7R/calendar-events/send-rsvp' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "event_id": "evt_abc123",
    "calendar_id": "cal_xyz789",
    "grant_id": "grant_def456",
    "status": "yes"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

## Overview

Retrieve a list of all agents and their associated run details.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Response

Returns an array of agent objects.

<ResponseField name="id" type="integer">
  The agent ID.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  When the agent was created.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the agent.
</ResponseField>

<ResponseField name="description" type="string">
  Internal description of the agent's function.
</ResponseField>

<ResponseField name="model_host" type="string">
  The model host used by the agent.
</ResponseField>

<ResponseField name="model" type="string">
  The model name.
</ResponseField>

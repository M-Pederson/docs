---
title: "Staging Table"
description: "Functions for managing pending evidence that hasn't yet cleared the KNOWS creation floor"
---

The `staging_evidence` Xano table holds weak evidence that arrives alone and doesn't clear the [evidence floor](/guides/known/documentation#evidence-model) by itself. When more evidence for the same pair arrives later, the cron combines staged rows with the new evidence, re-evaluates the floor, and — if it clears — creates the `KNOWS` edge with the full combined history.

## Critical rule

**Every evidence ingest pipeline MUST check staging for the same pair before creating or updating a `KNOWS` edge, regardless of whether the new evidence alone clears the floor.** Skipping the staging check on strong-source ingests would orphan previously-staged weak evidence. See the [LinkedIn → phone-import worked example](/guides/known/documentation#example-3-linkedin-phone-import-retroactive-backdating).

---

## `known/staging/insert_pending_evidence`

Writes a row to `staging_evidence` when evidence arrives that does not clear the floor alone.

**Inputs:** `person_a_id`, `person_b_id` (canonicalized), `source`, `weight`, `first_seen_at`, `evidence_occurred_at`, `raw_metadata`
**Outputs:** `staging_row_id`

## `known/staging/lookup_pending_for_pair`

Returns all staged rows for a canonicalized `(a, b)` pair. Called by every evidence ingest path **before** the `KNOWS` create/update step.

**Inputs:** `person_a_id`, `person_b_id`
**Outputs:** array of staged evidence rows

## `known/staging/consume_pending_for_pair`

Deletes all staged rows for a pair after they have been merged into a `KNOWS` edge. Called immediately after a `KNOWS` edge is created from combined staged + new evidence.

**Inputs:** `person_a_id`, `person_b_id`
**Outputs:** count of deleted rows

## `known/staging/expire_old_pending` (cron)

Deletes staged rows older than 730 days that never received a second matching signal. Bounds the staging table size.

**Cadence:** Weekly
**Tunable parameter:** `STAGING_EXPIRY_DAYS` (default 730)

---
title: "Get Collaborator Comments"
api: "GET https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborator-comments"
description: "Query all comments on a shared note."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField query="my_note_id" type="integer" required>
  The note ID to list comments for.
</ParamField>

<ResponseField name="items" type="array">
  List of comment records.
  <Expandable>
    <ResponseField name="id" type="integer">Comment ID.</ResponseField>
    <ResponseField name="user_id" type="integer">Author user ID.</ResponseField>
    <ResponseField name="my_note_id" type="integer">Associated note ID.</ResponseField>
    <ResponseField name="comment" type="string">Comment text.</ResponseField>
    <ResponseField name="created_at" type="timestamp">When the comment was created.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborator-comments?my_note_id=1' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

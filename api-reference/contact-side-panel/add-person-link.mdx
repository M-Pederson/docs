---
title: "Add Person Link"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:iTL91sBE/add/my_person/link"
description: "Add a link (URL) to a person record."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="link_url" type="string" required>
  The URL to add.
</ParamField>
<ParamField body="my_person_id" type="integer" required>
  The person ID.
</ParamField>

<ResponseField name="id" type="integer">Record ID.</ResponseField>
<ResponseField name="my_person_id" type="integer">Person ID.</ResponseField>
<ResponseField name="master_link_id" type="integer">Master link record ID.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iTL91sBE/add/my_person/link' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "link_url": "https://linkedin.com/in/johndoe",
    "my_person_id": 1
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "created_at": 1700000000,
  "my_person_id": 1,
  "master_link_id": 10
}
```
</ResponseExample>

description: 'Update a collection record'
---

## Overview

Updates an existing collection's title, description, icon, or color.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_id" type="integer" required>
  The ID of the collection to update.
</ParamField>

## Request Body

<ParamField body="title" type="string" required>
  The updated title of the collection.
</ParamField>

<ParamField body="description" type="string">
  The updated description.
</ParamField>

<ParamField body="icon_name" type="string">
  Updated icon name.
</ParamField>

<ParamField body="custom_icon_url" type="string">
  Updated custom icon URL.
</ParamField>

<ParamField body="collection_color" type="string">
  Updated hex color code.
</ParamField>

## Response

Returns the updated collection record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collections/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "title": "Updated Prospects",
    "description": "Updated description",
    "collection_color": "#3366FF"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "title": "Updated Prospects",
  "description": "Updated description",
  "collection_color": "#3366FF"
}
```
</ResponseExample>

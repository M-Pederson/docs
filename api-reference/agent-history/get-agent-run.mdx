## Overview

Retrieve the details of a specific agent run, including the steps taken and tools used during execution.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField body="agent_run_id" type="integer" required>
  The ID of the agent run to retrieve.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The agent run ID.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  When the agent run was created.
</ResponseField>

<ResponseField name="user" type="integer">
  The user who initiated the agent run.
</ResponseField>

<ResponseField name="agent_id" type="integer">
  The ID of the agent that was run.
</ResponseField>

<ResponseField name="args" type="object">
  Input arguments for the agent run.
</ResponseField>

<ResponseField name="final_response_text" type="string">
  The final response text from the agent run.
</ResponseField>

<ResponseField name="finish_reason" type="string">
  The reason the agent run finished (e.g., success, error).
</ResponseField>

<ResponseField name="total_usage" type="object">
  Usage statistics for the agent run.
</ResponseField>

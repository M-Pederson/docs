---
title: "Admin: Fix Dub Link"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/admin/fix-dub-link"
description: "Admin: Fix/recreate Dub link for a specific entry."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

### Body Parameters

<ParamField body="x_admin_key" type="string" required>
  Admin API key.
</ParamField>

<ParamField body="email" type="string" required>
  Email of the entry to fix.
</ParamField>

### Response

<ResponseField name="success" type="string">
  Indicates the fix was successful.
</ResponseField>

<ResponseField name="email" type="string">
  Email of the fixed entry.
</ResponseField>

<ResponseField name="referral_code" type="string">
  The referral code for the entry.
</ResponseField>

<ResponseField name="referral_link" type="string">
  The recreated referral link.
</ResponseField>

<ResponseField name="dub_link_id" type="string">
  The new Dub link ID.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/admin/fix-dub-link' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -d '{
    "x_admin_key": "YOUR_ADMIN_KEY",
    "email": "user@example.com"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": "true",
  "email": "user@example.com",
  "referral_code": "ORB-A7X2B",

  "dub_link_id": "dub_abc123"
}
```
</ResponseExample>

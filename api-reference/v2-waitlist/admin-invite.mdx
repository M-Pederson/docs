---
title: "Admin: Invite User"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/admin/invite"
description: "Admin: Invite a user from the waitlist."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

### Body Parameters

<ParamField body="x_admin_key" type="string" required>
  Admin API key.
</ParamField>

<ParamField body="email" type="string" required>
  Email of the user to invite.
</ParamField>

### Response

<ResponseField name="success" type="string">
  Indicates the invitation was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable result message.
</ResponseField>

<ResponseField name="email" type="string">
  Email of the invited user.
</ResponseField>

<ResponseField name="referral_count" type="string">
  Number of referrals the user made.
</ResponseField>

<ResponseField name="invited_at" type="string">
  Timestamp when the user was invited.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/admin/invite' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -d '{
    "x_admin_key": "YOUR_ADMIN_KEY",
    "email": "user@example.com"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": "true",
  "message": "User has been invited",
  "email": "user@example.com",
  "referral_count": "3",

```
</ResponseExample>

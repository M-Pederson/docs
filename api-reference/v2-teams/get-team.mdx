## Overview

Get a specific team by its ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="team_id" type="integer" required>
  The ID of the team to retrieve.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The team ID.
</ResponseField>

<ResponseField name="title" type="string">
  The team name.
</ResponseField>

<ResponseField name="description" type="string">
  The team description.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams/1" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "title": "Engineering",
  "description": "Engineering team"
}
```
</ResponseExample>

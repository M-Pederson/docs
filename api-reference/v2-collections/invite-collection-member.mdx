## Overview

Invites a team to access an existing collection by creating a collection membership.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="collection_id" type="integer">
  The ID of the collection to invite the team to.
</ParamField>

<ParamField body="team_id" type="integer" required>
  The ID of the team to invite.
</ParamField>

## Response

Returns the created collection membership record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-members \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_id": 1,
    "team_id": 5
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "team_id": 5,
  "status": "pending"
}
```
</ResponseExample>

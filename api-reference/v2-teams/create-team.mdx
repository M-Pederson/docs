---
title: 'Create Team'
api: 'POST https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams'
description: 'Create a team with members'
---

## Overview

Create a new team with optional members, icon, and color settings.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="title" type="string" required>
  The team name.
</ParamField>

<ParamField body="description" type="string">
  A description of the team.
</ParamField>

<ParamField body="icon_name" type="string">
  Icon name for the team.
</ParamField>

<ParamField body="custom_icon_url" type="string">
  URL for a custom team icon.
</ParamField>

<ParamField body="team_color" type="string">
  Hex color for the team.
</ParamField>

<ParamField body="user_ids" type="integer[]">
  Array of user IDs to add as members.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the created team.
</ResponseField>

<ResponseField name="title" type="string">
  The team name.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "title": "Engineering",
    "description": "Engineering team",
    "icon_name": "code",
    "team_color": "#4F46E5",
    "user_ids": [1, 2, 3]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "title": "Engineering"
}
```
</ResponseExample>

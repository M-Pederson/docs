## Overview

Retrieve the current system prompt configuration for the copilot. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Response

<ResponseField name="prompt" type="object">
  The system prompt object.
  <Expandable title="prompt properties">
    <ResponseField name="id" type="integer">
      The prompt ID.
    </ResponseField>
    <ResponseField name="created_at" type="timestamp">
      When the prompt was created.
    </ResponseField>
    <ResponseField name="name" type="string">
      The name of the prompt.
    </ResponseField>
    <ResponseField name="prompt" type="string">
      The prompt text content.
    </ResponseField>
    <ResponseField name="updated_at" type="timestamp">
      When the prompt was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:6pGFSEtq/system-prompt \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1"
```
</RequestExample>

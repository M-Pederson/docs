## Overview

Returns a single master company record by its ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="master_company_id" type="integer" required>
  The ID of the master company to retrieve.
</ParamField>

## Response

Returns the master company record.

```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:10Em-CSF/master-companies/100 \

  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Overview

Merge two contacts with user-edited data. Requires `merge_suspect_id` and `merged_data` JSON input.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="merge_suspect_id" type="integer" required>
  ID of the merge suspect record to merge.
</ParamField>

<ParamField body="merged_data" type="json" required>
  User-edited merged data to use for the merge.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the merge was successful.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://xh2o-yths-38lt.n7c.xano.io/api:6VkM3aHx/merge-suspects/merge-edited" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "merge_suspect_id": 123,
    "merged_data": {
      "name": "John Doe",
      "email": "john@example.com"
    }
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

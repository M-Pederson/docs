---
title: "Add Collaborator Comment"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborator-comments"
description: "Add a comment to a shared note."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="my_note_id" type="integer" required>
  The note to comment on.
</ParamField>
<ParamField body="comment" type="string" required>
  The comment text.
</ParamField>

<ResponseField name="id" type="integer">Comment ID.</ResponseField>
<ResponseField name="user_id" type="integer">Author user ID.</ResponseField>
<ResponseField name="comment" type="string">The comment text.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborator-comments' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "my_note_id": 1,
    "comment": "Great insight here!"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "created_at": 1700000000,
  "user_id": 5,
  "my_note_id": 1,
  "comment": "Great insight here!"
}
```
</ResponseExample>

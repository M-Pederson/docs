## Outcome Types

All Leverage Loops that help someone with something specific and all Outcomes have one of these types chosen during the AI chat interview process.

1. Hire key talent
2. Prospect for customers / clients
3. Find investors
4. Find investment opportunities / deal flow
5. Identify strategic partnerships
6. Make a purchase (non investment)
7. Find a co-founder / business partner
8. Find a mentor / advisor / board member
9. Find a job / new role
10. Find collaborators (research, content, projects)
11. Find media / PR / journalists
12. Find acquisition targets / acquirers
13. Find speakers / get speaking opportunities
14. Get advice / consultation _(catch-all for off-pattern asks)_

## Return Count vs. Scoring

The AI chat interview process always captures a return count.

How the requested count maps to the SQL query:

- **6 results** → `LIMIT 6` ordered by score (always returns 6, even if some are weak matches)
- **As many results as possible** → score threshold (e.g., `WHERE score > 0.7`) — count varies based on quality
- **Specific number of results** → `LIMIT N` ordered by score (always returns N regardless of confidence)

**Tradeoff:** A fixed `LIMIT` always returns the requested count but may include low-confidence matches. Combining both (`WHERE score > threshold ORDER BY score LIMIT N`) preserves quality but may return fewer than N if not enough candidates pass the threshold.

## Leverage Loops

Leverage Loops only suggest PEOPLE.

### Create a Leverage Loop to help a person with something specific

<div style={{padding: '16px', border: '2px solid #6B9FE8', borderRadius: '8px', color: '#fff'}}>

**Agent Flow:**
1. Interview User with AI chat to create a Leverage Loop Summary
2. Create three "synthetic profiles" of a person who could help this person with the specific need
3. Compare those "synthetic profiles" with everyone in User's profile using pre-filters and scoring
4. Reason "WHY"s for top scores
5. Deliver the number of results requested and/or confirmed in the AI chat interview

</div>

_The "specific need" maps to an [Outcome Type](#outcome-types) from the list above — but framed for the **target person**, not the User. (e.g., User wants to help target person hire key talent, find investors, find a co-founder, etc.)_

### Create a Leverage Loop

<div style={{padding: '16px', border: '2px solid #6B9FE8', borderRadius: '8px', color: '#fff'}}>

**Agent Flow:**
1. Create 3 "synthetic profile" of people that would be of interest to the target person
2. Compare that "synthetic profile" with everyone in User's profile using pre-filters & scoring
3. Reason "WHY"s for top scores
4. Deliver 6 results

</div>

## Outcomes

Outcomes can suggest any entity type: People, Companies, Live Events, etc

<div style={{padding: '16px', border: '2px solid #6B9FE8', borderRadius: '8px', color: '#fff'}}>

**Agent Flow:**
1. Interview User with AI chat to create an Outcome Summary
2. Define the type of entity the user wants — person, company, people & companies, live events, etc — use the entity schema to confirm and/or present options to User
3. Ask User to confirm number of desired results
   - 6 results
   - As many results as possible
   - Specific number of results

   _(Agent now knows TYPE of result User wants and number of results to deliver)_
4. Create 3 "synthetic profiles" of people that help the User achieve the Outcome
5. Compare those "synthetic profiles" with everyone in User's profile using pre-filters and scoring
6. Deliver the number of results requested and/or confirmed in the AI chat interview with "WHY"s reasoned

</div>

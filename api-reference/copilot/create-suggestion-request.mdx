## Overview

Create a new suggestion request through the copilot interface. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Request Body

<ParamField body="copilot_mode" type="string">
  The copilot mode. One of `outcome`, `loop`, or `serendipity`. Defaults to `outcome`.
</ParamField>

<ParamField body="outcome_title" type="string">
  Title for the outcome suggestion.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The suggestion request ID.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  When the request was created.
</ResponseField>

<ResponseField name="copilot_mode" type="string">
  The copilot mode used.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the request.
</ResponseField>

<ResponseField name="current_step" type="string">
  The current step in the workflow.
</ResponseField>

<ResponseField name="progress" type="integer">
  The progress percentage.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:6pGFSEtq/suggestion_request \
  -H "Content-Type: application/json" \

  -H "X-Branch: v1" \

  }'
```

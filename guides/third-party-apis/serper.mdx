## What it is

[Serper](https://serper.dev) is a fast, affordable Google Search API that returns structured search results including organic results, knowledge graphs, and related searches.

## Why we use it

- **Google results** — returns real Google Search data, not a secondary index
- **Structured output** — clean JSON responses with organic results, snippets, and metadata
- **Fast and reliable** — low-latency responses with high uptime
- **Cost effective** — competitive pricing for high-volume search queries

## How we use it at Orbiter

Serper is one of our web search providers, used alongside Tavily in enrichment pipelines where we need to query the web for real-time information to supplement our data.

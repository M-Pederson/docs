---
title: "Update Theme"
api: "PATCH https://xh2o-yths-38lt.n7c.xano.io/api:n2ii7Rpf/settings/theme/{id}"
description: "Update the user's theme setting (dark or light)."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="theme" type="string" required>
  The theme to set. Accepted values: `dark`, `light`.
</ParamField>

<ResponseField name="result" type="object">
  Empty object on success.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH \
  https://xh2o-yths-38lt.n7c.xano.io/api:n2ii7Rpf/settings/theme/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: live" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "theme": "dark"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

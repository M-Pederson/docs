description: 'Update an existing travel event'
---

## Overview

Update an existing travel event with new details.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="travel_id" type="integer" required>
  The ID of the travel event to update.
</ParamField>

<ParamField body="departing_from" type="string" required>
  The departure location.
</ParamField>

<ParamField body="destination" type="string" required>
  The destination location.
</ParamField>

<ParamField body="start_date" type="date" required>
  The trip start date.
</ParamField>

<ParamField body="end_date" type="date">
  The trip end date.
</ParamField>

<ParamField body="note" type="string">
  Optional note about the trip.
</ParamField>

<ParamField body="share_to_all" type="boolean">
  Whether to share the trip with all users.
</ParamField>

<ParamField body="share_to_connections" type="boolean">
  Whether to share the trip with connections only.
</ParamField>

<ParamField body="groups" type="array">
  List of group IDs to share the trip with.
</ParamField>

<ParamField body="formatted_address" type="string" required>
  The formatted address of the destination.
</ParamField>

## Response

Returns the updated travel trip object.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:P-1RmkSI/travel/trip/123 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "travel_id": 123,
    "departing_from": "New York",
    "destination": "Los Angeles",
    "start_date": "2026-04-01",
    "end_date": "2026-04-07",
    "formatted_address": "Los Angeles, CA, USA"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 123
}
```
</ResponseExample>

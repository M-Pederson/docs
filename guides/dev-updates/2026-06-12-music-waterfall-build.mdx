## Notes for the next session (agent or human)

- **Xano workspace 3**, all music artifacts tagged `musicbrainz`, branch `v1`. Sandbox-first for anything new.
- **XanoScript authoring traps** learned this session are recorded in this repo's `CLAUDE.md` (§ XanoScript authoring traps) — read them before writing or patching any function. The worst ones: `function.run` targets silently store as empty strings if the callee doesn't exist yet, and inline expressions / escaped newlines can be silently rewritten by the serializer. **Always re-fetch stored XanoScript after create/update and diff it.**
- `vectors/create-vectors-string` (#4676) returns the embedding as a **string** `"[0.1,...]"`, not an array — splice it directly into `vecf32()`.
- FalkorDB: no `a:Label` predicates in WHERE (use `'Label' IN labels(a)`); Cypher built in lambdas must be space-joined or use single-`\n` joins; send-cypher returns errors as **strings** — check `typeof r === 'string'` before reading `.data`.
- Queue tier/depth defaults must use `first_notnull` (never `first_notempty`) so bootstrap-lane `0` values survive.

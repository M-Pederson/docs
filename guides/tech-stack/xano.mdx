## What it is

[Xano](https://xano.com) is a no-code/low-code backend platform that provides a visual API builder, database management, and serverless function execution. It uses a proprietary scripting language called XanoScript for defining business logic.

## Why we use it

- **Rapid API development** — visual function builder with branching, conditionals, loops, and try/catch
- **Built-in database** — PostgreSQL-backed tables with a visual schema editor, addons (joins), and query builder
- **Scalable serverless** — functions execute on managed infrastructure with auto-scaling
- **Environment management** — branches (`v1`, `staging`, `dev`) with promotion workflows
- **MCP integration** — full workspace management via the Xano MCP server for Claude Code

## How we use it at Orbiter

Xano is the backbone of Orbiter's server-side logic. All enrichment pipelines (person, company, film/TV, funding, expertise), graph operations (via `mvp/falkor/send-cypher`), LLM orchestration (OpenRouter calls), and API endpoints run as Xano functions in **workspace 3**. The database stores all master records, join tables, enrichment data, and audit logs.

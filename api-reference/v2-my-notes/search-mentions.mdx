<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField query="query" type="string">
  Search term to filter mentions.
</ParamField>
<ParamField query="my_note_id" type="integer">
  Note ID context for the mention search.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/mentions?query=acme' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN'
```
</RequestExample>

{
  "my_companies": [],
  "my_persons": []

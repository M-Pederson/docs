---
title: "Patch My Note"
api: "PATCH https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/{my_note_id}"
description: "Partially update a note record, commonly used to toggle between note and document types."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField path="my_note_id" type="integer" required>
  The ID of the note to patch.
</ParamField>

<ParamField body="title" type="string">
  Updated title.
</ParamField>
<ParamField body="note" type="string">
  Updated content.
</ParamField>
<ParamField body="is_document" type="boolean">
  Whether this is a document or a simple note.
</ParamField>

<ResponseField name="id" type="integer">
  The note ID.
</ResponseField>
<ResponseField name="title" type="string">
  Note title.
</ResponseField>
<ResponseField name="is_document" type="boolean">
  Whether the note is a document.
</ResponseField>
<ResponseField name="tiptap_reference" type="string">
  Tiptap reference for the document.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/1' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "is_document": true
  }'
```
</RequestExample>

```json 200
  "is_document": true,

}
```
</ResponseExample>

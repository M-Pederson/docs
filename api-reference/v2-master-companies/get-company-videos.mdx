## Overview

Returns all video records associated with a specific master company.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="master_company_id" type="integer" required>
  The ID of the master company to get videos for.
</ParamField>

## Response

Returns an array of video records tied to the company.

<RequestExample>
```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:10Em-CSF/master-companies/100/videos \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "id": 1,
    "title": "Acme Corp Product Demo",

    "published_at": 1700000000
  }

```
</ResponseExample>

## Overview

Adds a master company record to the authenticated user's personal company list (my_company).

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="master_company_id" type="integer" required>
  The ID of the master company to add to the user's company list.
</ParamField>

## Response

Returns the created my_company record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:8FwiLIQ3/my-companies \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "master_company_id": 100
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "user_id": 10,
  "master_company_id": 100
}
```
</ResponseExample>

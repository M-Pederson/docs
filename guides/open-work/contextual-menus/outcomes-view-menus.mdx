## Scope

Contextual menus in the Outcomes view should make the next useful action available without pulling the operator out of the outcome workflow.

## Left Panel

- Edit - (allow user to edit name used in left panel)
- Create Collection - this should work; it should create a collection of all suggestion cards using the name from the left panel
- Export CSV (feature in dev)
- Export PDF (feature in dev)
- Archive
- Delete

![Outcomes left panel contextual menu](/images/open-work/contextual-menus/outcomes-panel.png)

## Cards

- Add Contact (only show this IF user does not have this contact - my_person / contacts ID = null)
- Add to Collection ("Collection" should be capitalized - should open drop down of users collections)
- Create Leverage Loop (ONLY show this IF user HAS this contact - my_person / contacts ID = NOT null - feature in development)
- Add to my Horizon (ONLY if no path visible)
- Remove from my Horizon (ONLY if the person in on User's horizon)
- Similar
- Relativity
- Delete

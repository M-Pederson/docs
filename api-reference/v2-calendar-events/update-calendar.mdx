---
title: "Update Calendar Preferences"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:LR2ywW7R/calendars"
description: "User chooses which calendars are active and which are not."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

### Body Parameters

<ParamField body="calendar_id" type="integer" required>
  The ID of the calendar to update.
</ParamField>

<ParamField body="event_widget_preference" type="boolean" required>
  Whether to show events from this calendar in the widget.
</ParamField>

### Response

<ResponseField name="id" type="integer">
  Unique identifier for the calendar.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  Timestamp when the calendar was created.
</ResponseField>

<ResponseField name="updated_at" type="timestamp">
  Timestamp when the calendar was last updated.
</ResponseField>

<ResponseField name="user_id" type="integer">
  The user who owns this calendar.
</ResponseField>

<ResponseField name="external_id" type="string">
  External calendar identifier from the provider.
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the calendar.
</ResponseField>

<ResponseField name="timezone" type="string">
  Calendar timezone.
</ResponseField>

<ResponseField name="is_primary" type="boolean">
  Whether this is the primary calendar.
</ResponseField>

<ResponseField name="event_widget_preference" type="boolean">
  Whether events from this calendar appear in the widget.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:LR2ywW7R/calendars' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "calendar_id": 1,
    "event_widget_preference": true
  }'
```
</RequestExample>

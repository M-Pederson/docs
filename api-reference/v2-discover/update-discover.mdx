description: 'Update a discover record'

## Overview

Updates an existing discover record. Can modify the query text or archive status.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="discover_id" type="integer" required>
  The ID of the discover record to update.
</ParamField>

## Request Body

<ParamField body="query" type="string">
  The updated query string.
</ParamField>

<ParamField body="archived" type="boolean">
  Whether to archive or unarchive the record.
</ParamField>

## Response

Returns the updated discover record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:rraxfaG7/discover/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "archived": true
  }'
```
</RequestExample>

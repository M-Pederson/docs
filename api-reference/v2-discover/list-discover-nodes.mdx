## Overview

Returns a paginated list of all discover node records associated with a specific discover query.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="discover_id" type="integer" required>
  The ID of the discover query to get nodes for.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer" required>
  Number of items per page.
</ParamField>

<ParamField query="offset" type="integer">
  Number of items to skip.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of discover node records containing id, created_at, updated_at, discover_id, label, name, and various UUID fields for linked entities.
</ResponseField>

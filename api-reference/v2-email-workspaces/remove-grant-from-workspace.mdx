## Overview

Remove a nylas_grant (email account) from a workspace. Deletes the join record only -- does not delete the grant or workspace.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="email_workspace_id" type="integer" required>
  ID of the email workspace.
</ParamField>

<ParamField path="nylas_grant_id" type="integer" required>
  ID of the nylas_grant to remove.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the grant was removed successfully.
</ResponseField>

```bash cURL
curl -X DELETE "https://xh2o-yths-38lt.n7c.xano.io/api:dUJtZDlP/email-workspaces/1/grants/3" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
```

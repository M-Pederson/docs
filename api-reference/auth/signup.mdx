---

## Overview

Register a new user account and receive a Bearer token. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Request Body

<ParamField body="user_login_email" type="string" format="email">
  The email address for the new account.
</ParamField>

<ParamField body="password" type="string" format="password">
  The password for the new account.
</ParamField>

## Response

<ResponseField name="authToken" type="string">
  A JWT Bearer token to use for authenticated requests.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:qMCc0ojP/auth/signup \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -d '{
    "user_login_email": "newuser@example.com",
    "password": "secure-password"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "authToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```
</ResponseExample>

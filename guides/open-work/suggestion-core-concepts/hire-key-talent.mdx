## Description

This outcome type surfaces individual candidates for open roles. The User is hiring (or hiring on behalf of someone) and wants suggestions of specific people they should consider — typically for a defined role with experience requirements, location preferences, and cultural fit signals.

**Best for:** filling specific open headcount where the User can articulate the role.

**Adjacent outcomes:** [Find a co-founder](/guides/open-work/suggestion-core-concepts/find-cofounder) for equity/founder-level partnerships, [Find a mentor / advisor](/guides/open-work/suggestion-core-concepts/find-mentor-advisor) for board/advisor seats.

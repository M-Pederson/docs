## Overview

Retrieve the contact panel data for a specific master person. Returns enriched contact information for display.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField body="master_person_id" type="integer">
  The ID of the master person to get contact panel data for.
</ParamField>

## Response

Returns the contact panel data object.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:1C8L4Ng6/contact_panel?master_person_id=123" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

---
title: "Edit My Note"
api: "PUT https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/{my_note_id}"
description: "Edit an existing note including title, content, and mentions."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField path="my_note_id" type="integer" required>
  The ID of the note to edit.
</ParamField>

<ParamField body="title" type="string">
  Updated title of the note.
</ParamField>
<ParamField body="note" type="string">
  Updated content of the note.
</ParamField>
<ParamField body="my_person_mentions" type="array">
  Array of person IDs mentioned in the note.
</ParamField>
<ParamField body="my_company_mentions" type="array">
  Array of company IDs mentioned in the note.
</ParamField>

<ResponseField name="id" type="integer">
  The note ID.
</ResponseField>
<ResponseField name="updated_at" type="timestamp">
  Last updated timestamp.
</ResponseField>
<ResponseField name="user_id" type="integer">
  The user ID of the note owner.
</ResponseField>
<ResponseField name="title" type="string">
  Updated title.
</ResponseField>
<ResponseField name="note" type="string">
  Updated content.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-notes/1' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "title": "Updated Meeting Notes",
    "note": "Revised Q4 targets..."
  }'
```
</RequestExample>

---
title: "Add Note Collaborator"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborators"
description: "Share a note or document with another user."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="my_note_id" type="integer" required>
  The note to share.
</ParamField>
<ParamField body="user_id" type="integer" required>
  The user to share the note with.
</ParamField>
<ParamField body="permission_level" type="string" required>
  Access level: `viewer`, `editor`, or `commentor`.
</ParamField>

<ResponseField name="id" type="integer">Collaborator record ID.</ResponseField>
<ResponseField name="my_note_id" type="integer">The shared note ID.</ResponseField>
<ResponseField name="shared_with_user" type="integer">User ID the note is shared with.</ResponseField>
<ResponseField name="permission_level" type="string">The assigned permission level.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:iuc7DvAP/my-note-collaborators' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "my_note_id": 1,
    "user_id": 5,
    "permission_level": "editor"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "created_at": 1700000000,
  "updated_at": null,
  "my_note_id": 1,
  "shared_with_user": 5,
  "permission_level": "editor"
}
```
</ResponseExample>

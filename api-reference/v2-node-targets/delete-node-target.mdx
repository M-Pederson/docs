## Overview

Delete a node_target record by ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="node_target_id" type="integer" required>
  ID of the node_target to delete.
</ParamField>

## Request Body

<ParamField body="node_uuid" type="string" required>
  The UUID of the node target.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the deletion was successful.
</ResponseField>

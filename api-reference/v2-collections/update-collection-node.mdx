---

## Overview

Updates an existing collection node, such as changing its assignment.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="collection_node_id" type="integer" required>
  The ID of the collection node to update.
</ParamField>

## Request Body

<ParamField body="assigned_to" type="integer">
  User ID to assign the node to.
</ParamField>

## Response

Returns the updated collection node record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "assigned_to": 10
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "assigned_to": 10
}
```
</ResponseExample>

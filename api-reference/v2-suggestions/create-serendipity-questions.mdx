## Overview

Create a new serendipity question record for a given serendipity suggestion.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="serendipity_suggestion_id" type="integer" required>
  The ID of the serendipity suggestion this question belongs to.
</ParamField>

<ParamField body="query" type="string" required>
  The question text.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Whether to perform deep research for this question.
</ParamField>

<ParamField body="team_ids" type="array">
  List of team IDs to associate.
</ParamField>

<ParamField body="selected_nodes" type="array">
  List of selected node identifiers.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the newly created serendipity question.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:MkA4QsNh/serendipity-questions \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "serendipity_suggestion_id": 123,
    "query": "What unexpected connections exist?",
    "deep_research": false
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 456
}
```
</ResponseExample>

This page tracks the contextual-menu work across Orbiter's primary view surfaces.

<Note>
**DESIGN NOTE:** Use the Leverage Loops card as the design reference. All contextual menus should open to the right, and the trash/delete icon should always render in red.

On hover state, use the accent color for the icon or hover state.

</Note>

## calendar-webhook-parse-event — #2477

Triggered by the Nylas `event.created` webhook when a new calendar event is received. Logs the raw event payload to `log_webhook_nylas`, resolves the Nylas grant and calendar, handles recurring events by fetching or creating `master_event` records via the Nylas API, creates the `event` record, then iterates through all participants — resolving each to a `master_person` (creating one from the email address if needed), copying to `my_person`, and logging a `calendar_event_activity` with appropriate type (organizer vs. participant). Kicks off async primary email resolution for each participant.

        function.run "mvp/nylas/get-nylas-keys" as $keys
        db.get nylas_grant {
          field_name = "grant_id"
          field_value = $input.grant_id
        } as $nylas_grant

        db.get calendar {
          field_name = "external_id"
          field_value = $input.event.calendar_id
          output = ["id"]
        } as $calendar

          where = $db.event.external_id == $input.event.id && $db.event.calendar_id == $calendar.id

        conditional {

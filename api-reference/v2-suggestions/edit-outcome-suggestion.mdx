## Overview

Partially update an existing outcome suggestion record by ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField body="outcome_suggestion_id" type="integer" required>
  The ID of the outcome suggestion to edit.
</ParamField>

## Request Body

<ParamField body="outcome_suggestion__" type="object">
  The fields to update on the outcome suggestion record.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the updated outcome suggestion.
</ResponseField>

```bash cURL
curl -X PATCH https://xh2o-yths-38lt.n7c.xano.io/api:MkA4QsNh/outcome-suggestions/123 \

  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "outcome_suggestion__": {}
  }'
```
</RequestExample>

```json 200
{
  "id": 123
}
```
</ResponseExample>

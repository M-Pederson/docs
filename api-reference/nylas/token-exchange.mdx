---
title: "Token Exchange"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:s_Mv_Fpv/nylas-grants/token-exchange"
description: "Exchange an authorization code for a Nylas grant token."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="code" type="string" required>Authorization code from OAuth flow.</ParamField>
<ParamField body="state" type="string">OAuth state parameter.</ParamField>
<ParamField body="redirect_url" type="string">Redirect URL used in the OAuth flow.</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:s_Mv_Fpv/nylas-grants/token-exchange' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"code": "auth_code_abc123"}'
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

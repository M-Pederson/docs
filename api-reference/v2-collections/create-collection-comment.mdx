## Overview

Creates a new comment on an existing collection.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="comment" type="string">
  The comment text.
</ParamField>

<ParamField body="collection_id" type="integer">
  The ID of the collection to comment on.
</ParamField>

## Response

Returns the created comment record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-comments \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "comment": "Great leads in this collection!",
    "collection_id": 1
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "comment": "Great leads in this collection!",
  "user_id": 10,
  "created_at": 1700000000
}
```
</ResponseExample>

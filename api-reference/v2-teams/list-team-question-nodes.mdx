## Overview

Get all nodes for a team question with full person, company, funding, and film/tv enrichment. Mirrors serendipity-suggestion-nodes behavior.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="team_question_id" type="integer" required>
  The ID of the team question to get nodes for.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of records to return.
</ParamField>

<ParamField query="offset" type="integer">
  Number of records to skip.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of enriched team question node records.
</ResponseField>

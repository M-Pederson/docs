## Overview

Retrieve all travel events that are shared to the authenticated user, plus the user's own personal travel events.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Response

Returns an array of travel trip objects.

<RequestExample>
```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:P-1RmkSI/travel/trip \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
[]
```
</ResponseExample>

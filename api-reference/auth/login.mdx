## Overview

Authenticate an existing user and receive a Bearer token for subsequent API requests. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Request Body

<ParamField body="email" type="string" format="email">
  The user's email address.
</ParamField>

<ParamField body="password" type="string">
  The user's password.
</ParamField>

## Response

<ResponseField name="authToken" type="string">
  A JWT Bearer token to use for authenticated requests.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:qMCc0ojP/auth/login \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -d '{
    "email": "user@example.com",
    "password": "your-password"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "authToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```
</ResponseExample>

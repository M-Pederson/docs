## Scope

Contextual menus in Leverage Loops should support the operator while they help another person make progress.

## Left Panel

- Edit - (allow user to edit name used in left panel)
- Create Collection - this should work; it should create a collection of all suggestion cards using the name from the left panel
- Export CSV (feature in dev)
- Export PDF (feature in dev)
- Archive
- Delete

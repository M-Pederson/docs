---
title: "Join Waitlist"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/join"
description: "Join the Orbiter waitlist with optional referral code."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

### Body Parameters

<ParamField body="email" type="string" required>
  Email address to join the waitlist.
</ParamField>

<ParamField body="ref" type="string">
  Optional referral code (e.g. `ORB-A7X2B`).
</ParamField>

### Response

<ResponseField name="success" type="string">
  Indicates whether the join was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable message about the result.
</ResponseField>

<ResponseField name="position" type="string">
  The user's position in the waitlist.
</ResponseField>

<ResponseField name="referral_link" type="string">
  The user's unique referral link.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:g7_LZ7hE/join' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -d '{
    "email": "user@example.com",
    "ref": "ORB-A7X2B"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": "true",
  "message": "You have been added to the waitlist!",
  "position": "42",
  "referral_link": "https://orbiter.io/ref/ORB-B8Y3C"
}
```
</ResponseExample>

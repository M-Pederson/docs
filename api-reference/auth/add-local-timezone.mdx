---

## Overview

Gets the authenticated user's timezone from the browser and saves it to the user record. This endpoint requires authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="time_zone" type="string">
  The IANA timezone string (e.g. `America/New_York`, `Europe/London`).
</ParamField>

## Response

Returns a success response on completion.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:qMCc0ojP/add/local-timezone \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_AUTH_TOKEN" \
  -d '{
    "time_zone": "America/New_York"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

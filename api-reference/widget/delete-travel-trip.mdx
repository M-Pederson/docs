## Overview

Delete an existing travel event by its ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="travel_id" type="integer" required>
  The ID of the travel event to delete.
</ParamField>

## Response

Returns a success confirmation.

<RequestExample>
```bash cURL
curl -X DELETE https://xh2o-yths-38lt.n7c.xano.io/api:P-1RmkSI/travel/trip \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "travel_id": 123
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

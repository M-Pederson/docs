---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

### Body Parameters

<ParamField body="master_company_id" type="integer">
  The master company ID to associate with. Nullable.
</ParamField>

<ParamField body="master_person_id" type="integer">
  The master person ID to associate with. Nullable.
</ParamField>

<ParamField body="link_url" type="string" required>
  The URL of the link to add.
</ParamField>

### Response

Returns the created master link record.

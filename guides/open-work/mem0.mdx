Orbiter's AI chats (outcomes, leverage loops, meeting prep, and how-I-know) currently run memory through Zep inside the server-side OpenUI05 dispatch path. This section covers where that stands today and how we move to Mem0 as a retrieval layer for distilled memories — not a transcript store and not the canonical profile store.

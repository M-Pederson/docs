## Overview

Add a new node_target record.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="node_uuid" type="string" required>
  The UUID of the node to target.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the created node_target record.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://xh2o-yths-38lt.n7c.xano.io/api:u7HQsdcu/node-targets" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "node_uuid": "abc-123-def-456"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1
}
```
</ResponseExample>

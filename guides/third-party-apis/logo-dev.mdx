## What it is

[Logo.dev](https://logo.dev) is an API that returns high-quality company logos from a domain name. Logos are served as images via a simple URL pattern — no complex integration required.

## Why we use it

- **Domain-based lookup** — pass a company domain, get the logo back
- **High quality** — curated logo database with consistent sizing and formatting
- **Simple integration** — logo URL can be constructed directly without an API call
- **Covers long tail** — handles lesser-known companies, not just big brands

## How we use it at Orbiter

Logo.dev provides company logos throughout the Orbiter UI wherever a company entity is displayed. The logo URL is derived from the company's domain and rendered inline as the company's profile image.

Example URL pattern:
```
https://img.logo.dev/{company_domain}?token=YOUR_TOKEN
```

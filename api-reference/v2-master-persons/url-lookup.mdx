## Overview

Looks up or creates a new master person record from a profile URL (e.g. LinkedIn profile URL).

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="input_url" type="string">
  The profile URL to look up (e.g. a LinkedIn profile URL).
</ParamField>

## Response

Returns the master person record found or created from the URL.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:WKgay2AU/master-persons/url-lookup \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "input_url": "https://linkedin.com/in/johndoe"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 50,
  "full_name": "John Doe",
  "profile_url": "https://linkedin.com/in/johndoe"
}
```
</ResponseExample>

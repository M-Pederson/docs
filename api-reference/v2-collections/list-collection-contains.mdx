---

## Overview

Returns all collections that contain a specific person or company as a node.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="master_person_id" type="integer">
  The ID of the master person to find collections for.
</ParamField>

<ParamField query="master_company_id" type="integer">
  The ID of the master company to find collections for.
</ParamField>

## Response

Returns an array of collection records containing the specified person or company.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-contains?master_person_id=50" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "id": 1,
    "title": "Top Prospects",
    "collection_color": "#FF5733"
  }
]
```
</ResponseExample>

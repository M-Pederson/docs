## Overview

Retrieve a master company record for display in the graph viewer. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Query Parameters

<ParamField body="master_company_id" type="integer" required>
  The ID of the master company to view.
</ParamField>

<ParamField body="data_source" type="string" required>
  The data source identifier.
</ParamField>

## Response

Returns the master company graph data object.

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:g7J1o-FF/view-master-company/123?master_company_id=123&data_source=default" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

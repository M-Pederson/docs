---
title: 'Edit Master Address'

description: 'Edit a master address record'
---

## Overview

Updates an existing master address record by its ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="master_address_id" type="integer" required>
  The ID of the master address to edit.
</ParamField>

## Request Body

<ParamField body="address_label" type="string">
  A label for the address (e.g. "Headquarters", "Home").
</ParamField>

<ParamField body="address_line_1" type="string">
  First line of the address.
</ParamField>

<ParamField body="address_line_2" type="string">
  Second line of the address.
</ParamField>

<ParamField body="city" type="string">
  City name.
</ParamField>

<ParamField body="state_region" type="string">
  State or region.
</ParamField>

<ParamField body="postal_code" type="string">
  Postal or ZIP code.
</ParamField>

<ParamField body="country" type="string">
  Country name.
</ParamField>

<ParamField body="address_note" type="string" required>
  A note about the address.
</ParamField>

## Response

Returns the updated master address record.

<RequestExample>
```bash cURL
curl -X PUT https://xh2o-yths-38lt.n7c.xano.io/api:Lnhgpj_M/master-addresses/1 \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "address_label": "Main Office",
    "address_line_1": "123 Main St",
    "city": "New York",
    "state_region": "NY",
    "postal_code": "10001",
    "country": "US",
    "address_note": "Updated headquarters address"
  }'
```
</RequestExample>

---
title: "Create Connect Request"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:8Pr6-Q01/orbiter-connect-requests"
description: "Ask another user to connect. If the other user has already requested to connect with you, auto-accept and create connection."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

### Body Parameters

<ParamField body="user_id" type="integer" required>
  The user ID you want to connect with.
</ParamField>

<ParamField body="request_message" type="string">
  Optional message to include with the connection request. Nullable.
</ParamField>

### Response

Returns the created connect request object.

<RequestExample>
```bash cURL
curl -X POST \
  'https://xh2o-yths-38lt.n7c.xano.io/api:8Pr6-Q01/orbiter-connect-requests' \
  -H 'Content-Type: application/json' \
  -H 'X-Data-Source: live' \
  -H 'X-Branch: v1' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "user_id": 55,
    "request_message": "Hi, I would like to connect with you!"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

api: 'PUT https://xh2o-yths-38lt.n7c.xano.io/api:dUJtZDlP/email-workspaces/{email_workspace_id}'
description: 'Update the display settings of an email workspace'
---

## Overview

Update the display settings (name, color, emoji) of an email workspace profile. Auth user must own the workspace.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="email_workspace_id" type="integer" required>
  ID of the workspace to update.
</ParamField>

## Request Body

<ParamField body="name" type="string">
  New display name.
</ParamField>

<ParamField body="color" type="string">
  New hex color (e.g. `#4F46E5`).
</ParamField>

<ParamField body="emoji" type="string">
  New emoji identifier.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the updated email workspace.
</ResponseField>

<ResponseField name="name" type="string">
  The updated display name.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT "https://xh2o-yths-38lt.n7c.xano.io/api:dUJtZDlP/email-workspaces/1" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "name": "Updated Workspace",
    "color": "#10B981",
    "emoji": "envelope"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "name": "Updated Workspace",
  "color": "#10B981",
  "emoji": "envelope"
}
```
</ResponseExample>

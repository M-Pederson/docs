## Overview

Retrieve all outcome action records for a given outcome suggestion.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField body="outcome_suggestion_id" type="integer" required>
  The ID of the outcome suggestion to get actions for.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of outcome action records.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://xh2o-yths-38lt.n7c.xano.io/api:MkA4QsNh/outcome-actions?outcome_suggestion_id=123" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "items": []
}
```
</ResponseExample>

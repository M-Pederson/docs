## What it is

[Tiptap](https://tiptap.dev) is a headless, framework-agnostic rich text editor built on ProseMirror. It provides a modular extension system for building custom editing experiences — from simple text inputs to full collaborative document editors.

## Why we use it

- **Headless architecture** — full control over UI and styling, no opinionated chrome
- **Extension system** — composable extensions for marks, nodes, and functionality
- **Collaboration** — real-time collaborative editing via Yjs or Tiptap Cloud
- **React integration** — first-class `@tiptap/react` bindings with SSR support
- **AI features** — built-in AI extension for content generation and editing assistance

## How we use it at Orbiter

Tiptap powers the rich text editing surfaces in Orbiter — content creation, notes, and collaborative editing. We use the React integration with Tailwind v4 styling and SSR-safe initialization patterns via TanStack Start.

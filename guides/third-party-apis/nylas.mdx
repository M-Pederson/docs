## What it is

[Nylas](https://www.nylas.com) is a universal API platform for email, calendar, and contacts. It provides a single API to connect to Gmail, Outlook, Exchange, and other providers, handling OAuth, sync, webhooks, and data normalization.

## Why we use it

- **Universal email access** — Gmail, Outlook, Exchange through one API
- **Calendar sync** — read/write calendar events across providers
- **Contact import** — pull contacts from connected email accounts
- **Webhooks** — real-time notifications for new messages, events, and contact changes
- **Grant management** — OAuth token lifecycle (exchange, refresh, disconnect)

## How we use it at Orbiter

Nylas powers Orbiter's email integration, calendar sync, and contact import features. Users connect their email accounts via Nylas OAuth, and Orbiter receives webhooks for new messages, calendar events, and contact changes. The Xano API has dedicated Nylas webhook endpoints and grant management functions.

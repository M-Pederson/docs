---

## Overview

Add one or more members to a team by providing their user IDs.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="team_id" type="integer" required>
  The ID of the team to add members to.
</ParamField>

<ParamField body="user_ids" type="integer[]" required>
  Array of user IDs to add as team members.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the members were added successfully.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/team-members" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "team_id": 1,
    "user_ids": [4, 5, 6]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

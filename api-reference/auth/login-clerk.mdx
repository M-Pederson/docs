## Overview

Authenticate a user via a Clerk session token and receive a Bearer token for subsequent API requests. This endpoint does not require authentication.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

## Request Body

<ParamField body="token" type="string" required>
  The Clerk session token.
</ParamField>

## Response

<ResponseField name="authToken" type="string">
  A JWT Bearer token to use for authenticated requests.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:qMCc0ojP/auth/login_clerk \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -d '{
    "token": "clerk-session-token"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "authToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```
</ResponseExample>

## Overview

Creates a new discover query record and initiates the analysis process. Optionally supports deep research mode and team sharing.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="query" type="string" required>
  The discover query to run.
</ParamField>

<ParamField body="discover_id" type="integer">
  An existing discover ID to continue from.
</ParamField>

<ParamField body="selected_nodes" type="array">
  Array of selected node identifiers to use as context.
</ParamField>

<ParamField body="team_ids" type="array">
  Array of team IDs to share the discover query with.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Enable deep research mode for more thorough analysis. Defaults to false.
</ParamField>

## Response

Returns the created discover record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:rraxfaG7/discover \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "query": "AI startups in fintech",
    "deep_research": true,
    "team_ids": [1, 2]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "query": "AI startups in fintech",
  "processing": true
}
```
</ResponseExample>

---

## Overview

Creates a new master address record associated with a company or person.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="master_company_id" type="integer">
  The ID of the master company to associate the address with.
</ParamField>

<ParamField body="master_person_id" type="integer">
  The ID of the master person to associate the address with.
</ParamField>

<ParamField body="address_label" type="string">
  A label for the address (e.g. "Headquarters", "Home").
</ParamField>

<ParamField body="formatted_address" type="string" required>
  The full formatted address string.
</ParamField>

<ParamField body="address_note" type="string">
  An optional note about the address.
</ParamField>

## Response

Returns the created master address record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:Lnhgpj_M/master-addresses \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "master_company_id": 100,
    "address_label": "Headquarters",
    "formatted_address": "123 Main St, New York, NY 10001",
    "address_note": "Main office"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "master_company_id": 100,
  "address_label": "Headquarters",
  "formatted_address": "123 Main St, New York, NY 10001",
  "address_note": "Main office"
}
```
</ResponseExample>

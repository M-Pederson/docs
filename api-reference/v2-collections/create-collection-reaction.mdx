## Overview

Creates an emoji reaction on a collection comment.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="collection_comment_id" type="integer">
  The ID of the comment to react to.
</ParamField>

<ParamField body="emoji" type="string" required>
  The emoji character for the reaction.
</ParamField>

## Response

Returns the created reaction record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-reactions \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_comment_id": 1,
    "emoji": "thumbsup"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_comment_id": 1,
  "emoji": "thumbsup",
  "user_id": 10
}
```
</ResponseExample>

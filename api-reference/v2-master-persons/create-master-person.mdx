---
title: 'Create Master Person'

description: 'Create a new master person record'
---

## Overview

Creates a new master person record with contact details and optional company association.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="full_name" type="string" required>
  The full name of the person.
</ParamField>

<ParamField body="email" type="array" required>
  Array of email addresses for the person.
</ParamField>

<ParamField body="company_domain" type="string">
  The domain of the person's company.
</ParamField>

<ParamField body="title" type="string">
  The person's job title.
</ParamField>

<ParamField body="note" type="string">
  A note about the person.
</ParamField>

<ParamField body="master_company_id" type="integer">
  The ID of the master company to associate with.
</ParamField>

<ParamField body="master_person_id" type="integer">
  An existing master person ID to link to.
</ParamField>

<ParamField body="profile_url" type="string">
  URL of the person's profile (e.g. LinkedIn).
</ParamField>

<ParamField body="company_name" type="string">
  Name of the person's company.
</ParamField>

<ParamField body="phone_array" type="object">
  JSON object containing phone number entries.
</ParamField>

## Response

Returns the created master person record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:WKgay2AU/master-persons \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "full_name": "Jane Smith",
    "email": ["jane@example.com"],
    "company_domain": "example.com",
    "title": "CTO",
    "note": "Met at conference",
    "master_company_id": 100,
    "profile_url": "https://linkedin.com/in/janesmith",
    "company_name": "Example Corp",
    "phone_array": {
      "phone": [
        {"number": "+1234567890", "type": "mobile"}
      ]
    }
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 51,
  "full_name": "Jane Smith",
  "master_company_id": 100
```

</ResponseExample>

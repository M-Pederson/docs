---
title: 'Create Collection'

description: 'Create a new collection with its relationships'
---

## Overview

Creates a new collection with optional initial nodes for persons, companies, funding rounds, and film/TV entities.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="title" type="string" required>
  The title of the collection.
</ParamField>

<ParamField body="description" type="string">
  A description of the collection.
</ParamField>

<ParamField body="icon_name" type="string">
  Name of the icon to display.
</ParamField>

<ParamField body="custom_icon_url" type="string">
  URL of a custom icon image.
</ParamField>

<ParamField body="collection_color" type="string" required>
  Hex color code for the collection.
</ParamField>

<ParamField body="team_id" type="integer">
  Team ID to share the collection with.
</ParamField>

<ParamField body="master_person_ids" type="array">
  Array of master person IDs to add as initial nodes.
</ParamField>

<ParamField body="master_company_ids" type="array">
  Array of master company IDs to add as initial nodes.
</ParamField>

<ParamField body="company_funding_round_ids" type="array">
  Array of company funding round IDs to add as initial nodes.
</ParamField>

<ParamField body="film_tv_ids" type="array">
  Array of film/TV IDs to add as initial nodes.
</ParamField>

<ParamField body="film_tv_award_ids" type="array">
  Array of film/TV award IDs to add as initial nodes.
</ParamField>

## Response

Returns the created collection record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collections \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "title": "Top Prospects",
    "description": "Key prospects for Q1",
    "collection_color": "#FF5733",
    "master_person_ids": [50, 51],
    "master_company_ids": [100]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "title": "Top Prospects",
  "description": "Key prospects for Q1",
  "collection_color": "#FF5733"
}
```
</ResponseExample>

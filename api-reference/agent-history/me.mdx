description: 'Get the record belonging to the authentication token'
---

## Overview

Retrieve the user record associated with the current authentication token.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The user ID.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  When the user was created.
</ResponseField>

<ResponseField name="name" type="string">
  The user's name.
</ResponseField>

<ResponseField name="email" type="string">
  The user's email address.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:5mQm30zj/auth/me \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "created_at": 1700000000000,
  "name": "John Doe",
  "email": "john@example.com"
}
```
</ResponseExample>

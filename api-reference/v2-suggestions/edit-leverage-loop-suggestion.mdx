## Overview

Partially update an existing leverage loop suggestion record by ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField body="leverage_loop_suggestion_id" type="integer" required>
  The ID of the leverage loop suggestion to edit.
</ParamField>

## Request Body

<ParamField body="leverage_loop_suggestion__" type="object">
  The fields to update on the leverage loop suggestion record.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the updated leverage loop suggestion.
</ResponseField>

```bash cURL
curl -X PATCH https://xh2o-yths-38lt.n7c.xano.io/api:MkA4QsNh/leverage-loop-suggestions/123 \

  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "leverage_loop_suggestion__": {}
  }'
```
</RequestExample>

```json 200
{
  "id": 123
}
```

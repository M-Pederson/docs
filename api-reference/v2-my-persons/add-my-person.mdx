## Overview

Adds an existing master person record to the authenticated user's personal contacts (my_person).

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="master_person_id" type="integer">
  The ID of the master person to add to the user's contacts.
</ParamField>

## Response

Returns the created my_person record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:Et3oQAtI/my-persons \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "master_person_id": 50
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "user_id": 10,
  "master_person_id": 50
}
```
</ResponseExample>

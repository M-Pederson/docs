All Cypher queries for creating Film & TV title nodes, IMDB credit edges (WORKED_ON), and Film & TV Award nodes with award edges (HAS_AWARD, WON_AWARD, NOMINATED_FOR).

---

## mvp/node/add-film-tv-node (function #2320)

Creates a Film_TV node for an IMDB title. The node gets a dynamic type label based on the title type (Movie, TV_Series, etc.).

```cypher
// Version v1
CREATE (n:Entity:Film_TV:{{$var.imdbType}} {
  uuid: randomUUID(),
  title: "{{$var.imdbTitle.title}}",
  imdb_url: "{{$var.imdbTitle.imdb_url}}",
  type: "{{$var.imdbTitle.type}}",
  description: "{{$var.imdbTitle.description}}",
  genres: "{{$var.imdbTitle.genres|join(',')}}",
  year: "{{$var.imdbTitle.year}}",
  imdb_rating: "{{$var.imdbTitle.imdb_rating}}",
  running_time: "{{$var.imdbTitle.running_time}}",
  mpaa_rating: "{{$var.imdbTitle.mpaa_rating}}",
  countries: "{{$var.imdbTitle.countries|join(',')}}",
  languages: "{{$var.imdbTitle.languages|join(',')}}",
  name_embedding: vecf32({{$var.embeddingArray}})
})
RETURN n.uuid AS uuid
```

## mvp/edges/add-imdb-credit-edges (function #10514)

Creates WORKED_ON edges between a Person and a Film_TV node for each credit. Weight is dynamically assigned based on role seniority (director=6 through default crew=15).

```cypher
// Version v1
MATCH (p:Person {uuid: '{{ $var.filmCredit.person_node_uuid }}'})
MATCH (f:Film_TV {uuid: '{{ $var.filmCredit.film_tv_node_uuid }}'})
MERGE (p)-[r:WORKED_ON]->(f)
SET r.roles = '{{ $var.filmCredit.roles | join(", ") }}',
    r.position = '{{ $var.filmCredit.position }}',
    r.description = '{{ $var.filmCredit.person.name }} worked on
      {{ $var.filmCredit.film_tv.title }} with credits for
      {{ $var.filmCredit.roles | join(" and ") }}',
    r.weight = {% if $var.filmCredit.position in ['director', 'co-director']
      or 'director' in $var.filmCredit.roles
      or 'co-director' in $var.filmCredit.roles %}6
    {% elseif $var.filmCredit.position in ['showrunner', 'creator']
      or 'showrunner' in $var.filmCredit.roles
      or 'creator' in $var.filmCredit.roles %}7
    {% elseif $var.filmCredit.position in ['writer', 'co-writer',
      'screenwriter', 'screenplay', 'teleplay', 'written by', 'story by']
      or 'writer' in $var.filmCredit.roles
      or 'screenwriter' in $var.filmCredit.roles
      or 'screenplay' in $var.filmCredit.roles
      or 'teleplay' in $var.filmCredit.roles
      or 'written by' in $var.filmCredit.roles
      or 'story by' in $var.filmCredit.roles %}8
    {% elseif $var.filmCredit.position in ['cinematographer',
      'director of photography', 'dop', 'dp', 'editor', 'film editor',
      'composer', 'music by', 'original score', 'original music',
      'production designer', 'production design']
      or 'cinematographer' in $var.filmCredit.roles
      or 'director of photography' in $var.filmCredit.roles
      or 'composer' in $var.filmCredit.roles
      or 'production designer' in $var.filmCredit.roles %}9
    {% elseif $var.filmCredit.position in ['producer', 'co-producer']
      or 'producer' in $var.filmCredit.roles
      or 'co-producer' in $var.filmCredit.roles %}10
    {% elseif $var.filmCredit.position in ['costume designer',
      'costume design', 'vfx supervisor', 'visual effects supervisor',
      'vfx', 'stunt coordinator', 'stunts',
      '2nd unit director', 'second unit director']
      or 'costume designer' in $var.filmCredit.roles
      or 'vfx supervisor' in $var.filmCredit.roles
      or 'stunt coordinator' in $var.filmCredit.roles %}11
    {% elseif $var.filmCredit.position in ['executive producer',
      'co-executive producer']
      or 'executive producer' in $var.filmCredit.roles
      or 'co-executive producer' in $var.filmCredit.roles %}12
    {% elseif $var.filmCredit.position in ['supervising producer',
      'consulting producer']
      or 'supervising producer' in $var.filmCredit.roles
      or 'consulting producer' in $var.filmCredit.roles %}13
    {% elseif $var.filmCredit.position in ['associate producer',
      'line producer']
      or 'associate producer' in $var.filmCredit.roles
      or 'line producer' in $var.filmCredit.roles %}14
    {% else %}15{% endif %},
    r.imdb_person_url = '{{ $var.filmCredit.imdb_person_url }}',
    r.imdb_title_url = '{{ $var.filmCredit.imdb_title_url }}',
    r.created_at = timestamp()
RETURN '{{ $var.filmCredit.person.name }}' AS person,
       type(r) AS relationship,
       '{{ $var.filmCredit.film_tv.title }}' AS title,
       r.roles AS roles, r.weight AS weight,
       r.description AS description
```

## mvp/edge/add-node-edge-imdb-award (function #2334)

Creates Film_TV_Award nodes and all award-related edges. This is a single large Cypher that handles: MERGE Film_TV node (with dynamic type label), optional WORKED_ON edge, MERGE Film_TV_Award node (with Winner/Nominee label), HAS_AWARD edge, and WON_AWARD or NOMINATED_FOR edge.

Uses parameterized queries (`$p_*` params) to prevent injection.

### MERGE Film_TV + Award Node + All Edges

```cypher
// Version v1
{% if $var.filmTvAward.person is not null
  and $var.filmTvAward.person.node_uuid is not null
  and $var.filmTvAward.person.node_uuid != '' %}
MATCH (p:Person { uuid: $p_person_uuid })
{% endif %}

MERGE (f:Film_TV { imdb_url: $p_imdb_url })
ON CREATE SET
  f.uuid = randomUUID(),
  f.created_at = $p_now,
  f.display_label = "Film_TV",
  f.title = $p_title,
  f.type = $p_type,
  {% if $var.filmTV.tagline is not null and $var.filmTV.tagline != '' %}
  f.tagline = $p_tagline,
  {% endif %}
  f.description = $p_description,
  f.genres = $p_genres,
  f.year = $p_year,
  {% if $var.filmTV.imdb_rating is not null %}
  f.imdb_rating = $p_imdb_rating,
  {% endif %}
  {% if $var.filmTV.running_time is not null %}
  f.running_time = $p_running_time,
  {% endif %}
  {% if $var.filmTV.mpaa_rating is not null %}
  f.mpaa_rating = $p_mpaa_rating,
  {% endif %}
  f.countries = $p_countries,
  f.languages = $p_languages,
  {% if $var.filmTV.image_url is not null %}
  f.image_url = $p_image_url,
  {% endif %}
  f.source = $p_source
ON MATCH SET
  f.updated_at = $p_now,
  f.title = $p_title,
  f.description = $p_description,
  {% if $var.filmTV.imdb_rating is not null %}
  f.imdb_rating = $p_imdb_rating,
  {% endif %}
  {% if $var.filmTV.image_url is not null %}
  f.image_url = $p_image_url,
  {% endif %}
  f.source = $p_source

-- Dynamic type label based on film type
{% if $var.filmTV.type == 'movie' %}
SET f:Movie
{% elseif $var.filmTV.type == 'tvSeries' or $var.filmTV.type == 'tv_series' %}
SET f:TV_Series
{% elseif $var.filmTV.type == 'tvMovie' or $var.filmTV.type == 'tv_movie' %}
SET f:TV_Movie
{% elseif $var.filmTV.type == 'tvMiniSeries' or $var.filmTV.type == 'tv_mini_series' %}
SET f:TV_Mini_Series
{% elseif $var.filmTV.type == 'short' %}
SET f:Short_Film
{% elseif $var.filmTV.type == 'documentary' %}
SET f:Documentary
{% elseif $var.filmTV.type == 'tvSpecial' or $var.filmTV.type == 'tv_special' %}
SET f:TV_Special
{% endif %}

-- Optional WORKED_ON edge (only if person context exists)
{% if $var.filmTvAward.person is not null
  and $var.filmTvAward.person.node_uuid is not null %}
WITH p, f
MERGE (p)-[w:WORKED_ON]->(f)
ON CREATE SET
  w.created_at = $p_now,
  w.valid_from = $p_valid_from,
  {% if $var.creditRole is not null and $var.creditRole != '' %}
  w.credit_role = $p_credit_role,
  {% endif %}
  w.weight = 0.5,
  w.source = $p_source
ON MATCH SET
  w.updated_at = $p_now,
  w.weight = 0.5,
  w.source = $p_source
WITH p, f
{% else %}
WITH f
{% endif %}

-- MERGE Film_TV_Award node
MERGE (a:Film_TV_Award {
  year: $p_award_year,
  award_type: $p_award_type,
  award_description: $p_award_description,
  film_tv_imdb_url: $p_project_imdb_url
})
ON CREATE SET
  a.uuid = randomUUID(),
  a.created_at = $p_now,
  a.display_label = "Film_TV_Award",
  a.description = $p_award_text,
  a.nominee_or_winner = $p_nominee_or_winner,
  {% if $var.filmTvAward.shared_with is not null
    and $var.filmTvAward.shared_with|length > 0 %}
  a.shared_with = $p_shared_with,
  {% endif %}
  a.project_title = $p_project_title,
  a.name_embedding = vecf32({{$var.awardEmbeddingArray}})
ON MATCH SET
  a.updated_at = $p_now,
  a.description = $p_award_text,
  a.nominee_or_winner = $p_nominee_or_winner,
  {% if $var.filmTvAward.shared_with is not null
    and $var.filmTvAward.shared_with|length > 0 %}
  a.shared_with = $p_shared_with,
  {% endif %}
  a.project_title = $p_project_title,
  a.name_embedding = vecf32({{$var.awardEmbeddingArray}})

-- Dynamic label: Winner or Nominee
{% if $var.filmTvAward.nominee_or_winner == 'Winner' %}
SET a:Winner
{% else %}
SET a:Nominee
{% endif %}

-- HAS_AWARD edge from Film_TV to Award
MERGE (f)-[h:HAS_AWARD]->(a)
ON CREATE SET
  h.created_at = $p_now,
  h.valid_from = $p_valid_from,
  h.weight = 0.5,
  h.source = $p_source
ON MATCH SET
  h.updated_at = $p_now,
  h.weight = 0.5,
  h.source = $p_source

-- Person award edge: WON_AWARD or NOMINATED_FOR
{% if $var.filmTvAward.person is not null
  and $var.filmTvAward.person.node_uuid is not null %}
WITH p, a
{% if $var.filmTvAward.nominee_or_winner == 'Winner' %}
MERGE (p)-[r:WON_AWARD]->(a)
ON CREATE SET
  r.created_at = $p_now,
  r.valid_from = $p_valid_from,
  r.valid_to = $p_valid_to,
  r.weight = 0.3,
  r.description = $p_edge_description,
  r.source = $p_source
ON MATCH SET
  r.updated_at = $p_now,
  r.weight = 0.3,
  r.description = $p_edge_description,
  r.source = $p_source
{% else %}
MERGE (p)-[r:NOMINATED_FOR]->(a)
ON CREATE SET
  r.created_at = $p_now,
  r.valid_from = $p_valid_from,
  r.valid_to = $p_valid_to,
  r.weight = 0.7,
  r.description = $p_edge_description,
  r.source = $p_source
ON MATCH SET
  r.updated_at = $p_now,
  r.weight = 0.7,
  r.description = $p_edge_description,
  r.source = $p_source
{% endif %}
{% endif %}

RETURN a.uuid AS uuid
```

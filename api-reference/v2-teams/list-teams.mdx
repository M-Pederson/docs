## Overview

Get all teams for the authenticated user with optional search and pagination.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Query Parameters

<ParamField query="query" type="string">
  Search term to filter teams.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="offset" type="integer">
  Number of records to skip.
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of records to return.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of team records.
</ResponseField>

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "items": []
}
```
</ResponseExample>

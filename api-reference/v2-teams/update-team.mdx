---
title: 'Update Team'
api: 'PUT https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams/{team_id}'
description: 'Update a team record'
---

## Overview

Update a team's title, description, icon, and color settings.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="team_id" type="integer" required>
  The ID of the team to update.
</ParamField>

## Request Body

<ParamField body="title" type="string" required>
  The updated team name.
</ParamField>

<ParamField body="description" type="string">
  The updated team description.
</ParamField>

<ParamField body="icon_name" type="string">
  Updated icon name.
</ParamField>

<ParamField body="custom_icon_url" type="string">
  Updated custom icon URL.
</ParamField>

<ParamField body="team_color" type="string">
  Updated hex color.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The team ID.
</ResponseField>

<ResponseField name="title" type="string">
  The updated team name.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams/1" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "title": "Engineering v2",
    "description": "Updated engineering team",
    "team_color": "#10B981"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "title": "Engineering v2"
}
```
</ResponseExample>

---
title: "Create Date Descriptor"
api: "POST https://xh2o-yths-38lt.n7c.xano.io/api:mGnfV_UB/date-descriptors"
description: "Add or get a date descriptor."
---

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>
<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>
<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

<ParamField body="descriptor" type="string" required>
  The descriptor text (e.g. "Birthday", "Anniversary").
</ParamField>

<ResponseField name="id" type="integer">
  The date descriptor ID.
</ResponseField>
<ResponseField name="created_at" type="number">
  Creation timestamp.
</ResponseField>
<ResponseField name="descriptor" type="string">
  The descriptor text.
</ResponseField>
<ResponseField name="user_id" type="integer">
  The user who created it, or null for system descriptors.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  https://xh2o-yths-38lt.n7c.xano.io/api:mGnfV_UB/date-descriptors \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: live" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "descriptor": "Wedding Anniversary"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 5,
  "created_at": 1700000000000,
  "descriptor": "Wedding Anniversary",
  "user_id": 1
}
```
</ResponseExample>

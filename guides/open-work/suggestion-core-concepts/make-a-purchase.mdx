## Description

This outcome type surfaces vendors, sellers, or service providers for purchases the User wants to make for use — not financial return. Could be services (coaches, lawyers, agencies), products (software, high-end goods), or assets (homes, art, equipment). The User wants vetted recommendations, not generic search results.

**Best for:** high-trust purchases where word-of-mouth and reputation matter more than reviews.

**Adjacent outcomes:** [Get advice / consultation](/guides/open-work/suggestion-core-concepts/get-advice) when the User isn't sure what to buy yet, [Find a mentor / advisor](/guides/open-work/suggestion-core-concepts/find-mentor-advisor) when the relationship is ongoing instead of transactional.

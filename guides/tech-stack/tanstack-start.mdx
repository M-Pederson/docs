## What it is

[TanStack Start](https://tanstack.com/start) is a full-stack React framework built on top of TanStack Router and TanStack Query. It provides server-side rendering, type-safe file-based routing, server functions via `createServerFn`, and first-class deployment support for Cloudflare Workers.

## Why we use it

- **Type-safe routing** — file-based routes with full TypeScript inference, no runtime route mismatches
- **Server functions** — `createServerFn` for secure server-side logic without a separate API layer
- **SSR + hydration** — server-rendered pages that hydrate seamlessly on the client
- **TanStack Query integration** — data fetching, caching, and SSR prefetching built in
- **Cloudflare Workers deployment** — edge-first, globally distributed

## How we use it at Orbiter

TanStack Start powers the Orbiter.io web application frontend. Server functions handle authenticated data fetching from the Xano backend, and TanStack Query manages client-side cache invalidation and optimistic updates.

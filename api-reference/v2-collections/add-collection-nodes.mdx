title: 'Add Collection Nodes'
api: 'POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes'
description: 'Bulk add multiple nodes to a collection'
---

## Overview

Adds multiple nodes (persons, companies, funding rounds, film/TV entities) to an existing collection in a single request.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="collection_id" type="integer" required>
  The ID of the collection to add nodes to.
</ParamField>

<ParamField body="master_person_ids" type="array">
  Array of master person IDs to add.
</ParamField>

<ParamField body="master_company_ids" type="array">
  Array of master company IDs to add.
</ParamField>

<ParamField body="company_funding_round_ids" type="array">
  Array of company funding round IDs to add.
</ParamField>

<ParamField body="film_tv_ids" type="array">
  Array of film/TV IDs to add.
</ParamField>

<ParamField body="film_tv_award_ids" type="array">
  Array of film/TV award IDs to add.
</ParamField>

## Response

Returns the created collection node records.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-nodes \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_id": 1,
    "master_person_ids": [50, 51],
    "master_company_ids": [100]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "collection_id": 1,
  "nodes_added": 3
}
```
</ResponseExample>

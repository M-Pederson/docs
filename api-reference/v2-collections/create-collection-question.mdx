## Overview

Creates a new question on a collection to run analysis against its nodes.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="collection_id" type="integer" required>
  The ID of the collection to add the question to.
</ParamField>

<ParamField body="query" type="string" required>
  The question text.
</ParamField>

<ParamField body="selected_nodes" type="array">
  Array of selected node identifiers to use as context.
</ParamField>

<ParamField body="team_ids" type="array">
  Array of team IDs.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Enable deep research mode. Defaults to false.
</ParamField>

## Response

Returns the created question record.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:U_4ewelb/collection-questions \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "collection_id": 1,
    "query": "Who are the top executives?",
    "deep_research": false
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "collection_id": 1,
  "query": "Who are the top executives?"
}
```
</ResponseExample>

## Overview

Retrieve all connection requests from the user and invitations to the user, for display in the Connection widget.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Response

Returns an object containing connection requests and invitations.

<RequestExample>
```bash cURL
curl -X GET https://xh2o-yths-38lt.n7c.xano.io/api:P-1RmkSI/connections/orbiter_connections \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

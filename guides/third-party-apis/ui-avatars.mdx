## What it is

[UI Avatars](https://ui-avatars.com) is a simple API that generates letter-based avatar images from a person's name initials. No API key required — avatars are generated via URL parameters.

## Why we use it

- **Zero setup** — no API key, no auth, just a URL
- **Deterministic** — same name always produces the same avatar
- **Customizable** — size, colors, font, rounding, and background via URL params
- **Fast CDN** — images served from a global CDN

## How we use it at Orbiter

UI Avatars provides fallback profile pictures when a person or company doesn't have a photo available. The avatar URL is constructed from the entity's name initials and rendered inline wherever a profile image would normally appear.

Example URL pattern:
```
https://ui-avatars.com/api/?name=John+Doe&size=128&background=6B9FE8&color=fff&rounded=true
```

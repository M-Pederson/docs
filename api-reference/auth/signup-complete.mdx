## Overview

Called by the frontend to notify the backend that the user has completed the full signup process. This endpoint requires authentication.

<Note>
  This endpoint is typically called automatically by the frontend after a user finishes onboarding. It does not accept any request body parameters.
</Note>

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Response

Returns an empty object on success.

<RequestExample>
```bash cURL
curl -X POST https://xh2o-yths-38lt.n7c.xano.io/api:qMCc0ojP/auth/signup-complete \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_AUTH_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{}
```
</ResponseExample>

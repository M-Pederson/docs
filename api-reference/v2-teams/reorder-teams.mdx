description: 'Reorder the authenticated user teams'
---

## Overview

Reorder the auth user's teams by submitting an ordered array of team IDs.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="team_ids" type="integer[]">
  Ordered array of team IDs representing the desired display order.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the reorder was successful.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/teams/reorder" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "team_ids": [3, 1, 2]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

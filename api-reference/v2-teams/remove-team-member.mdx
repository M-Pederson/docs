## Overview

Remove a member from a team.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField path="team_member_id" type="integer" required>
  The ID of the team member record to remove.
</ParamField>

## Request Body

<ParamField body="team_id" type="integer" required>
  The ID of the team.
</ParamField>

<ParamField body="user_id" type="integer" required>
  The ID of the user to remove.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the removal was successful.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://xh2o-yths-38lt.n7c.xano.io/api:idMy4x2v/team-members/1" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "team_id": 1,
    "user_id": 5
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true
}
```
</ResponseExample>

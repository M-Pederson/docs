## What it is

[Groq](https://groq.com) is an AI inference platform built on custom LPU (Language Processing Unit) hardware, delivering extremely fast token generation speeds. It provides an OpenAI-compatible API for running open-source and proprietary models at low latency.

## Why we use it

- **Speed** — sub-second response times for most queries, significantly faster than GPU-based inference
- **OpenAI-compatible API** — same request format as OpenRouter/OpenAI, easy to swap in
- **Compound models** — Groq's compound models (e.g. `groq/compound`) chain multiple models together for complex reasoning tasks
- **Open-source model hosting** — fast inference for Meta Llama, Alibaba Qwen, and other open models
- **Cost efficient** — competitive pricing for high-throughput workloads

## How we use it at Orbiter

Groq is available as an inference provider through our pipeline for workloads where speed is the priority over model capability. Currently using `groq/compound` for multi-step reasoning tasks that benefit from Groq's model chaining approach.

Available models include:
- **Groq:** `groq/compound`, `groq/compound-mini`
- **Meta Llama:** `llama-3.1-8b-instant`, `llama-3.3-70b-versatile`, `meta-llama/llama-4-scout-17b-16e-instruct`
- **Alibaba Cloud:** `qwen/qwen3-32b`
- **OpenAI (hosted on Groq):** `whisper-large-v3`, `whisper-large-v3-turbo`

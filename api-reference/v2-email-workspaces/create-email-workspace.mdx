---
title: 'Create Email Workspace'
api: 'POST https://xh2o-yths-38lt.n7c.xano.io/api:dUJtZDlP/email-workspaces'
description: 'Create a new email workspace with one or more nylas_grant accounts'
---

## Overview

Create a new email workspace with one or more nylas_grant accounts. Position auto-increments. Provider is comma-separated unique values e.g. google or google,microsoft.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Display name for the email workspace.
</ParamField>

<ParamField body="color" type="string">
  Hex color for the workspace (e.g. `#4F46E5`).
</ParamField>

<ParamField body="emoji" type="string">
  Emoji identifier for the workspace.
</ParamField>

<ParamField body="icon" type="string">
  Icon identifier for the workspace.
</ParamField>

<ParamField body="nylas_grant_ids" type="integer[]">
  Array of nylas_grant IDs to link to this workspace.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the created email workspace.
</ResponseField>

<ResponseField name="name" type="string">
  The display name of the workspace.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://xh2o-yths-38lt.n7c.xano.io/api:dUJtZDlP/email-workspaces" \
  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "name": "Work Email",
    "color": "#4F46E5",
    "emoji": "briefcase",
    "nylas_grant_ids": [1, 2]
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "name": "Work Email",
  "color": "#4F46E5",
  "emoji": "briefcase"
}
```
</ResponseExample>

## Overview

Partially update an existing serendipity suggestion record by ID.

## Headers

<ParamField header="X-Data-Source" type="string" required>
  Identifies the data source for the request.
</ParamField>

<ParamField header="X-Branch" type="string" required>
  The Xano branch to target (e.g. `v1`, `staging`, `dev`).
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login or signup.
</ParamField>

## Path Parameters

<ParamField body="serendipity_suggestion_id" type="integer" required>
  The ID of the serendipity suggestion to edit.
</ParamField>

## Request Body

<ParamField body="serendipity_suggestion__" type="object">
  The fields to update on the serendipity suggestion record.
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The ID of the updated serendipity suggestion.
</ResponseField>

```bash cURL

  -H "Content-Type: application/json" \
  -H "X-Data-Source: YOUR_DATA_SOURCE" \
  -H "X-Branch: v1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "serendipity_suggestion__": {}
  }'
```
</RequestExample>

```json 200
{
  "id": 123
}
```

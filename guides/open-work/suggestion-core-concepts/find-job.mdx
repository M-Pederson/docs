## Description

This outcome type is the inverse of [Hire key talent](/guides/open-work/suggestion-core-concepts/hire-key-talent) — the User is the candidate. The system surfaces companies and hiring decision-makers (founders, hiring managers, VPs) at organizations that match the User's target profile.

**Best for:** senior IC and exec career moves where applying through a public portal is the wrong path.

**Adjacent outcomes:** [Hire key talent](/guides/open-work/suggestion-core-concepts/hire-key-talent) for the inverse (the hirer's side), [Find a co-founder](/guides/open-work/suggestion-core-concepts/find-cofounder) when the move is into equity-level partnership.
